Ниже приведен список часто используемых команд в Linux с кратким описанием:

### Управление файлами и директориями:
1. **ls** — выводит список файлов и директорий.
2. **cd** — смена текущей директории.
3. **pwd** — показывает полный путь до текущей директории.
4. **mkdir** — создание новой директории.
5. **rmdir** — удаление пустой директории.
6. **rm** — удаление файлов или директорий.
7. **cp** — копирование файлов или директорий.
8. **mv** — перемещение или переименование файлов или директорий.
9. **touch** — создание пустого файла.
10. **cat** — вывод содержимого файла.
11. **more** — постраничный просмотр содержимого файла.
12. **less** — аналогично more, но с возможностью перемотки назад.
13. **head** — вывод первых строк файла.
14. **tail** — вывод последних строк файла.
15. **find** — поиск файлов и директорий по заданным критериям.
16. **locate** — поиск файла по имени.

### Работа с архивами и сжатием:
1. **tar** — создание и извлечение архивов.
2. **gzip** — сжатие файлов.
3. **gunzip** — распаковка файлов сжатых gzip.
4. **zip** — создание zip архива.
5. **unzip** — распаковка zip архива.

### Управление правами доступа:
1. **chmod** — изменение прав доступа к файлам и директориям.
2. **chown** — изменение владельца файла или директории.
3. **chgrp** — изменение группы файла или директории.

### Работа с процессами:
1. **ps** — вывод списка процессов.
2. **top** — интерактивное отображение текущих процессов.
3. **kill** — завершение процесса по его ID.
4. **killall** — завершение всех процессов по имени.
5. **bg** — отправка процесса в фоновый режим.
6. **fg** — возврат процесса в активный режим.
7. **jobs** — вывод списка запущенных задач.

### Работа с сетью:
1. **ifconfig** — настройка и просмотр сетевых интерфейсов.
2. **ping** — проверка доступности узла в сети.
3. **netstat** — вывод сетевых подключений и статистики.
4. **wget** — загрузка файлов по сети.
5. **curl** — передача данных с или на сервер.
6. **ssh** — подключение к удалённой системе через SSH.
7. **scp** — копирование файлов между хостами по SSH.
8. **ftp** — клиент для работы с FTP серверами.

### Утилиты и системная информация:
1. **df** — отображение информации о дисковом пространстве.
2. **du** — оценка использования дискового пространства.
3. **free** — информация о доступной памяти.
4. **uname** — информация о системе.
5. **uptime** — вывод времени работы системы.
6. **whoami** — вывод текущего пользователя.
7. **hostname** — вывод имени хоста.
8. **dmesg** — вывод системных сообщений ядра.
9. **history** — вывод истории команд.

### Управление пакетами (для разных дистрибутивов):
1. **apt** — менеджер пакетов для Debian/Ubuntu.
2. **yum** — менеджер пакетов для RHEL/CentOS.
3. **dnf** — менеджер пакетов для Fedora.
4. **pacman** — менеджер пакетов для Arch Linux.
5. **snap** — управление пакетами snap.
6. **dpkg** — низкоуровневый менеджер пакетов для Debian.

### Перенаправление ввода/вывода и пайпы:
1. **>** — перенаправление вывода в файл (с перезаписью).
2. **>>** — перенаправление вывода в файл (с добавлением).
3. **<** — перенаправление ввода из файла.
4. **|** — передача вывода одной команды как ввода в другую.

Этот список включает основные команды, но Linux предоставляет намного больше возможностей и инструментов для управления системой.
