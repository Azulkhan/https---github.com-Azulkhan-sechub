Вот примеры работы с XML с использованием пространства имен в Lua и JavaScript:

Lua:
```lua
local xml = require("xml")

-- XML с пространством имен
local xml_str = [[
<root xmlns:aawp="v1:awp">
  <aawp:ncalayer>Текст элемента</aawp:ncalayer>
</root>
]]

local doc = xml.eval(xml_str)

-- Получение содержимого элемента
local ncalayer_text = doc:find("aawp:ncalayer"):text()
print(ncalayer_text)
```

JavaScript:
```javascript
const { DOMParser } = require('xmldom');

// XML с пространством имен
const xmlStr = `
<root xmlns:aawp="v1:awp">
  <aawp:ncalayer>Текст элемента</aawp:ncalayer>
</root>
`;

const parser = new DOMParser();
const xmlDoc = parser.parseFromString(xmlStr, "text/xml");

// Получение содержимого элемента
const ncalayerText = xmlDoc.getElementsByTagNameNS("v1:awp", "ncalayer")[0].textContent;
console.log(ncalayerText);
```

Обратите внимание, что в Lua используется библиотека `xml`, а в JavaScript используется стандартный модуль `xmldom` для работы с XML.
