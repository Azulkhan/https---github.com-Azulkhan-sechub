В AngularJS для отправки POST-запроса с заголовками XML и использованием прокси вы можете сделать следующее:

```javascript
$http({
  method: 'POST',
  url: 'your_api_url',
  data: 'your_xml_data',
  headers: {
    'Content-Type': 'application/xml' // Установка заголовка Content-Type на application/xml
  },
  proxy: {
    host: 'your_proxy_host',
    port: 'your_proxy_port'
  }
}).then(function successCallback(response) {
    // Обработка успешного ответа
  }, function errorCallback(response) {
    // Обработка ошибки
});
```

Замените `'your_api_url'` на URL вашего API, `'your_xml_data'` на ваш XML-документ, `'your_proxy_host'` и `'your_proxy_port'` на хост и порт вашего прокси-сервера соответственно.
