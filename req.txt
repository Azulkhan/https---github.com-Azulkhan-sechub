В Lua нет прямой функции для открытия ключей .p12. Однако вы можете использовать внешние библиотеки или вызывать команды системы через `os.execute`, чтобы выполнить это. Например, вы можете использовать openssl с помощью команды `os.execute` в Lua:

```lua
os.execute("openssl pkcs12 -in your_key.p12 -out extracted_key.pem -nodes")
```

Это извлечет закрытый ключ из вашего файла .p12 в файл `extracted_key.pem`.

Если вы хотите узнать, какие библиотеки уже установлены на вашей системе, вам может помочь менеджер пакетов или список установленных пакетов.

1. **Linux**:
   - Для Debian-подобных систем (например, Ubuntu) вы можете использовать команду `dpkg --get-selections | grep -v deinstall` для просмотра установленных пакетов.
   - Для Red Hat-подобных систем (например, Fedora) вы можете использовать команду `rpm -qa` для просмотра установленных пакетов.

2. **macOS**:
   - Вы можете использовать команду `brew list` для просмотра установленных пакетов, если вы используете Homebrew.

3. **Windows**:
   - Вы можете просмотреть список установленных программ через "Установка и удаление программ" в панели управления.

После того как вы получите список установленных пакетов, вы можете проанализировать его, чтобы определить, какие библиотеки уже доступны на вашей системе.
