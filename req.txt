Чтобы сделать так, чтобы не было повторений и брались только первые значения в шаблоне Go (Go templates), можно использовать комбинацию мапы и if-условия для проверки, было ли значение уже использовано. Вот пример, как это можно сделать:

```go
{{- $seen := dict -}}
{{- range $item_key, $item_value := .items -}}
    {{- if not (index $seen $item_value) -}}
        {{- $seen = set $seen $item_value true -}}
        {{- /* Ваш код для обработки уникальных значений */ -}}
        {{- $item_value -}}
    {{- end -}}
{{- end -}}
```

В этом примере:
1. Создаётся пустой словарь `$seen` для отслеживания уже встреченных значений.
2. Внутри цикла `range` проверяется, есть ли уже значение `$item_value` в `$seen`.
3. Если значения ещё нет в словаре, то:
    - Добавляется значение в словарь `$seen`.
    - Выполняется ваш код для обработки уникальных значений.

Таким образом, каждый элемент будет обработан только один раз, и повторяющиеся значения будут игнорироваться.
