В HTML атрибут `colspan` принимает только целые числа, поэтому установка значения `1.5` не будет работать. Атрибут `colspan` указывает, на сколько столбцов должна распространяться ячейка таблицы, и должен быть целым числом.

Если нужно сделать что-то, что кажется подобным "полуторному" colspan, вам придется использовать CSS для управления шириной столбцов и ячеек таблицы.

Пример:

```html
<style>
    .half-colspan {
        width: 150%; /* Устанавливаем ширину ячейки на 150% */
    }
</style>

<table border="1">
    <tr>
        <td>Cell 1</td>
        <td class="half-colspan" colspan="2">Cell 2 (1.5 colspan visual)</td>
        <td>Cell 3</td>
    </tr>
</table>
```

В этом примере `colspan="2"` используется, чтобы объединить два столбца, а класс `half-colspan` задает ширину ячейки на 150% от стандартной ширины одного столбца. Это не полноценное решение, но визуально может приблизить к требуемому результату.
