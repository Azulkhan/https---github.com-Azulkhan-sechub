В Lua вы можете убрать значения из строки, используя различные методы. Если вам нужно удалить одно значение спереди и два в конце строки, вы можете использовать следующий код:

```lua
local str = "Ваша строка с данными"
-- Удаление одного значения спереди
str = str:sub(2)

-- Удаление двух значений с конца
str = str:sub(1, -3)

print(str) -- Вывод результата
```

Этот код использует функцию `sub`, чтобы удалить части строки с заданными индексами.
