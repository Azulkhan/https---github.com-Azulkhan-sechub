
Вот пример Lua скрипта, который вызывает одну и ту же Java функцию с разными данными:

```lua
-- Подключение библиотеки Java
local luaj = require("luaj")

-- Путь к Java классу
local classPath = "com.example.MyJavaClass"

-- Функция для вызова Java метода с разными данными
local function callJavaMethod(data)
    -- Создание объекта Java
    local javaObject, error = luaj.newInstance(classPath)
    if not javaObject then
        print("Ошибка создания объекта Java:", error)
        return
    end
    
    -- Вызов Java метода с передачей данных
    local success, result = luaj.callMethod(javaObject, "myJavaMethod", {data})
    if success then
        print("Результат вызова Java метода:", result)
    else
        print("Ошибка вызова Java метода:", result)
    end
end

-- Примеры данных для передачи в Java метод
local data1 = "первые данные"
local data2 = "вторые данные"

-- Вызов функции с разными данными
callJavaMethod(data1)
callJavaMethod(data2)
```

Пожалуйста, замените `"com.example.MyJavaClass"` на полное имя вашего Java класса, а `"myJavaMethod"` на имя метода, который вы хотите вызвать.
