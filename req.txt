Понял, вы хотите сделать два изображения одинакового размера, при этом они имеют разные оригинальные размеры — одно 189x237, а другое 250x275. В этом случае, чтобы не растягивать или сжимать изображение, но привести их к одинаковому размеру, можно использовать подход с установкой контейнера фиксированного размера и масштабированием изображений внутри него.

Вот как это можно сделать:

### CSS:

```css
.image-container {
  width: 250px;   /* Ширина контейнера */
  height: 275px;  /* Высота контейнера */
  overflow: hidden; /* Прячет излишки изображения */
}

img {
  width: 100%;   /* Масштабирование изображения по ширине контейнера */
  height: auto;  /* Высота будет пропорциональной ширине */
}
```

### HTML:

```html
<div class="image-container">
  <img src="image1.jpg" alt="Первое изображение">
</div>
<div class="image-container">
  <img src="image2.jpg" alt="Второе изображение">
</div>
```

### Пояснение:
- **`.image-container`**: фиксированный контейнер, который задает одинаковый размер для всех изображений (например, 250x275).
- **`img`**: изображение масштабируется по ширине контейнера, а высота подстраивается автоматически. Если изображение превышает высоту контейнера, оно обрежется из-за свойства `overflow: hidden`.

Этот метод позволяет изображениям быть одинаковыми по размеру в рамках их контейнеров, но при этом они сохранят свои пропорции и не будут растягиваться.
