
Чтобы объявить глобальную переменную в Lua внутри функции, достаточно присвоить значение переменной без использования локального ключевого слова `local`. Глобальные переменные в Lua доступны в любом месте вашего скрипта после их объявления. Вот пример:

```lua
function setGlobal()
    globalVar = "Это глобальная переменная"
end

setGlobal()
print(globalVar) -- Вывод: Это глобальная переменная
```

В этом примере `globalVar` объявляется внутри функции `setGlobal` без ключевого слова `local`, что делает её глобальной. После вызова `setGlobal`, `globalVar` доступна для использования вне функции.

Однако, следует с осторожностью использовать глобальные переменные из-за возможности неожиданного изменения их значений, что может усложнить отладку программы. Лучше использовать локальные переменные, когда это возможно, и передавать переменные в функции через параметры для более чистого и понятного кода.
