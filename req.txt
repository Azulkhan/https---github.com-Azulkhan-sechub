
Вы можете получить название файла из объекта File, который представляет выбранный файл. Вот пример, как это можно сделать:

```html
<input type="file" id="fileInput" onchange="getFileDetails(event)">
<script>
    function getFileDetails(event) {
        const fileInput = event.target;
        const files = fileInput.files;
        for (let i = 0; i < files.length; i++) {
            const fileName = files[i].name;
            if (fileName.startsWith('auth-rsa')) {
                alert('Выбран файл с именем: ' + fileName);
                // Дополнительные действия с файлом...
            } else {
                alert('Выберите файлы, имена которых начинаются с "auth-rsa".');
                fileInput.value = ''; // Очистить выбранные файлы
                return;
            }
        }
    }
</script>
```

Этот код выводит название файла, если оно начинается с "auth-rsa". Вы можете добавить в эту функцию любую логику, которая должна выполняться с выбранным файлом.
