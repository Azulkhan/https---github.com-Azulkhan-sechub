
Для отправки POST-запроса на HTTPS-ссылку из Lua скрипта, вы можете использовать библиотеку `LuaSocket` вместе с `LuaSec` для поддержки протокола HTTPS. Вот пример кода, который делает POST-запрос:

```lua
-- Подключаем библиотеки
local https = require("ssl.https")
local ltn12 = require("ltn12")

-- Тело запроса
local body = "param1=value1&param2=value2" -- Замените на свои данные

-- Заголовки запроса
local headers = {
    ["Content-Type"] = "application/x-www-form-urlencoded",
    ["Content-Length"] = string.len(body)
}

-- Функция для отправки POST запроса
local function sendPostRequest(url, body, headers)
    local response = {}
    local result, statusCode, responseHeaders, statusText = https.request{
        method = "POST",
        url = url,
        source = ltn12.source.string(body),
        headers = headers,
        sink = ltn12.sink.table(response)
    }

    return table.concat(response), result, statusCode, responseHeaders, statusText
end

-- Замените URL на ваш HTTPS-адрес
local url = "https://yourhttpsurl.com"

-- Выполняем запрос
local response, result, statusCode, responseHeaders, statusText = sendPostRequest(url, body, headers)

-- Выводим результат
print("Response:", response)
print("Status Code:", statusCode)
print("Status Text:", statusText)
```

Перед использованием этого кода убедитесь, что у вас установлены и корректно настроены Lua, LuaSocket и LuaSec. Установку этих библиотек можно выполнить через менеджер пакетов LuaRocks или другие средства, в зависимости от вашей системы и предпочтений.
