
В вашем AngularJS приложении вы можете использовать объект `window`, чтобы получить доступ к глобальным переменным и объектам, включая библиотеки, которые вы загрузили. Вы можете использовать метод `Object.keys()` для получения списка всех свойств объекта `window` и проверить, какие библиотеки доступны.

Вот пример кода, который выводит список всех глобальных переменных и объектов, доступных в вашем AngularJS приложении:

```javascript
angular.module('yourApp').controller('YourController', function($scope) {
  $scope.listLibraries = function() {
    var libraries = Object.keys(window);
    console.log('Список доступных библиотек и переменных:');
    console.log(libraries);
  };
});
```

Вызовите функцию `$scope.listLibraries()` в вашем контроллере или в другом месте вашего кода AngularJS, чтобы получить список всех доступных библиотек и переменных. Вывод будет отображаться в консоли вашего браузера.
