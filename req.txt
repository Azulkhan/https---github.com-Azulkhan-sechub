
Я здесь, чтобы помочь, так что давайте посмотрим, как можно сохранить выбранную дату после использования ASPxCalendar. Вы можете сохранить выбранную дату в базе данных, сессии, куки или любом другом хранилище данных в зависимости от потребностей вашего приложения. Вот пример того, как можно сохранить выбранную дату в сессии:

```csharp
protected void ASPxCalendar1_SelectionChanged(object sender, EventArgs e)
{
    // Получаем выбранную дату
    DateTime selectedDate = ASPxCalendar1.SelectedDate;
    
    // Сохраняем выбранную дату в сессии
    Session["SelectedDate"] = selectedDate;
}
```

Если вам нужно сохранить эту дату в базе данных, пример может выглядеть следующим образом:

```csharp
protected void ASPxCalendar1_SelectionChanged(object sender, EventArgs e)
{
    // Получаем выбранную дату
    DateTime selectedDate = ASPxCalendar1.SelectedDate;
    
    // Здесь код для подключения к базе данных и сохранения даты
    // Пример использования псевдокода для иллюстрации
    string connectionString = "..."; // Ваша строка подключения
    using (SqlConnection connection = new SqlConnection(connectionString))
    {
        string query = "INSERT INTO YourTableName (YourDateColumnName) VALUES (@SelectedDate)";
        using (SqlCommand command = new SqlCommand(query, connection))
        {
            command.Parameters.AddWithValue("@SelectedDate", selectedDate);
            connection.Open();
            command.ExecuteNonQuery();
        }
    }
}
```

Важно помнить, что вам нужно будет адаптировать этот код под вашу конкретную ситуацию, включая названия таблицы и столбцов, а также строку подключения к вашей базе данных.
