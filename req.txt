В `p-table` из PrimeNG атрибут `[value]` используется для передачи данных, которые вы хотите отобразить в таблице. Вместо `data` вам нужно передать переменную из вашего компонента Angular, которая содержит массив объектов, которые вы хотите отобразить в таблице.

Например, если у вас есть компонент Angular с массивом данных `tableData`, вы можете использовать его так:

```html
<p-table [value]="tableData">
  <!-- Здесь идут шаблоны заголовка, тела и подвала -->
</p-table>
```

В вашем компоненте TypeScript (`app.component.ts`) вы должны иметь что-то подобное:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  tableData = [
    { id: 1, name: 'Item 1', price: 100 },
    { id: 2, name: 'Item 2', price: 150 },
    { id: 3, name: 'Item 3', price: 200 }
  ];
}
```

Здесь `tableData` является массивом объектов, каждый из которых представляет одну строку данных в таблице. Вы можете использовать любое имя для вашей переменной, важно лишь, чтобы она содержала корректные данные для отображения в `p-table`.
