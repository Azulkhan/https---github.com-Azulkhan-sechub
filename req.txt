Чтобы сделать два изображения одинакового размера, можно воспользоваться CSS. Вот два способа:

### 1. Использование фиксированной ширины и высоты:
```css
img {
  width: 200px;  /* Задайте фиксированную ширину */
  height: 200px; /* Задайте фиксированную высоту */
  object-fit: cover; /* Масштабирует изображение и обрезает его */
}
```
Этот способ задает фиксированные размеры для всех изображений, и они будут подогнаны под эти размеры.

### 2. Масштабирование только по ширине или высоте:
Если вы хотите сохранить пропорции изображения, можно задать только ширину или высоту, а другой параметр подстроится автоматически:

```css
img {
  width: 200px;  /* Задайте ширину, высота подстроится автоматически */
  height: auto;
}
```

В зависимости от ситуации, вы можете выбрать подходящий способ, чтобы изображения выглядели одинаково.
