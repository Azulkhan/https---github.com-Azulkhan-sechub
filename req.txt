Чтобы разместить два изображения друг над другом, но чтобы одно из них перекрывало другое на 25%, можно использовать CSS с абсолютным позиционированием или отрицательными отступами. Вот пример:

### HTML:
```html
<div class="image-container">
    <img src="image1.jpg" alt="Image 1" class="image image-top">
    <img src="image2.jpg" alt="Image 2" class="image image-bottom">
</div>
```

### CSS:
```css
.image-container {
    position: relative;
    width: max-content; /* Можно задать фиксированную ширину */
}

.image {
    display: block;
    width: 300px; /* Задайте ширину изображений по вашему желанию */
}

.image-top {
    position: absolute;
    top: -25%; /* 25% перекрытие */
    left: 0;
    z-index: 1; /* Поставим верхнее изображение выше */
}

.image-bottom {
    position: relative;
    z-index: 0;
}
```

### Пояснение:
1. **Контейнер `.image-container`**: Этот элемент используется для того, чтобы разместить изображения друг над другом.
2. **Абсолютное позиционирование**: Для верхнего изображения используется `position: absolute`, чтобы сдвинуть его на 25% вверх через свойство `top: -25%`.
3. **Z-index**: Это свойство позволяет контролировать, какое изображение будет сверху. В данном случае верхнее изображение (`.image-top`) будет перекрывать нижнее на 25%.

Этот код создаст эффект, где одно изображение будет накладываться на другое на 25%, как если бы их тела перекрывали друг друга.
