Понял! Чтобы одно изображение стояло внизу справа и немного заходило верхним левым углом на другое изображение, можно использовать комбинацию относительного и абсолютного позиционирования. Вот пример:

### HTML:
```html
<div class="image-container">
    <img src="image1.jpg" alt="Image 1" class="image image-first">
    <img src="image2.jpg" alt="Image 2" class="image image-second">
</div>
```

### CSS:
```css
.image-container {
    position: relative;
    width: max-content; /* Устанавливаем контейнер под размер изображения */
}

.image {
    display: block;
    width: 300px; /* Ширина изображений, настройте под свои нужды */
}

.image-first {
    position: relative; /* Это изображение остается на месте */
}

.image-second {
    position: absolute;
    top: 75%; /* Сместите вниз на 75% от высоты первого изображения */
    left: 50%; /* Смещаем второе изображение вправо */
    transform: translate(-25%, -25%); /* Немного корректируем положение, чтобы угол заходил на первое изображение */
    z-index: 1; /* Это изображение будет выше первого */
}
```

### Пояснение:
1. **Контейнер `.image-container`**: Этот блок содержит оба изображения.
2. **Первое изображение (`.image-first`)** остается на месте и не смещается.
3. **Второе изображение (`.image-second`)** позиционируется абсолютно. Мы смещаем его вниз (`top: 75%`) и вправо (`left: 50%`), а затем с помощью `transform: translate(-25%, -25%)` сдвигаем так, чтобы его верхний левый угол немного заходил на первое изображение.
4. **`z-index: 1`** позволяет второму изображению быть выше первого.

Таким образом, верхний левый угол второго изображения будет немного накладываться на первое изображение, создавая эффект частичного перекрытия.
