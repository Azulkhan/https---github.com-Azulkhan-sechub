В PostgreSQL, `UNION ALL` используется для объединения результатов двух или более запросов в один набор данных, при этом дубликаты строк не удаляются (в отличие от `UNION`, который удаляет дубликаты). Вот пример использования `UNION ALL`:

```sql
SELECT column1, column2
FROM table1
UNION ALL
SELECT column1, column2
FROM table2;
```

В этом примере результаты запроса из `table1` и `table2` будут объединены в один набор данных без удаления дубликатов.
