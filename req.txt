
В AngularJS можно использовать `$timeout` сервис для установки временной задержки перед выполнением определенного кода. Вот пример:

```javascript
app.controller('MyController', function($scope, $timeout) {
    // Выполнить что-то через 3 секунды
    $timeout(function() {
        // ваш код здесь
    }, 3000);
});
```

Этот код выполнит функцию через 3 секунды после инициализации контроллера.
