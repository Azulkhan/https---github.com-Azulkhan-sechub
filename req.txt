В Lua вы можете вложить один цикл `for` в другой, чтобы пройтись по элементам двух списков или таблиц. Если вам нужно, чтобы определенные записи второго `for` соответствовали определенным критериям из первого, вы можете использовать условные операторы внутри вложенного цикла.

Вот пример, как это можно сделать:

```lua
-- Первый список
local list1 = {1, 2, 3}

-- Второй список
local list2 = {2, 3, 4, 5}

-- Два вложенных цикла for
for i = 1, #list1 do
    local value1 = list1[i]
    
    -- Второй цикл for
    for j = 1, #list2 do
        local value2 = list2[j]
        
        -- Условие: если value2 больше или равно value1
        if value2 >= value1 then
            print("Value1:", value1, "Value2:", value2)
        end
    end
end
```

В этом примере внешний цикл проходит по элементам `list1`, а внутренний — по элементам `list2`. Внутри внутреннего цикла используется условие, чтобы проверить, если значение из `list2` больше или равно значению из `list1`. Если условие выполнено, значения выводятся на экран.

Вы можете адаптировать этот код под свои нужды, изменяя условие или операции внутри вложенного цикла.
