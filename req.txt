В Oracle SQL для получения первого дня месяца можно использовать функцию `TRUNC` с указанием параметра `'MM'`, который усекает дату до начала месяца. Вот пример запроса:

```sql
SELECT TRUNC(SYSDATE, 'MM') AS first_day_of_month
FROM dual;
```

Этот запрос вернет первый день текущего месяца. Если необходимо получить первый день другого месяца, можно заменить `SYSDATE` на нужную дату.
