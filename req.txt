Для создания такого блока (`div`), который начинается с левого края, не доходит до правого и имеет закругленные углы, можно использовать CSS-свойства `width`, `border-radius` и `margin`. Вот пример кода:

```html
<div class="rounded-div">
  Контент блока
</div>
```

```css
.rounded-div {
  width: 80%; /* Ширина 80% от родительского элемента */
  margin-left: 0; /* Прижат к левому краю */
  margin-right: auto; /* Оставляет пустое пространство справа */
  border-radius: 15px; /* Закругление углов */
  background-color: lightblue; /* Цвет фона для примера */
  padding: 20px; /* Отступы внутри блока */
}
```

Этот код создаёт блок, который занимает 80% ширины контейнера, прижат к левому краю и имеет закругленные углы радиусом 15px. Вы можете настроить ширину и радиус углов по своему усмотрению.
