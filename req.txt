Да, с помощью OpenSSL можно извлечь данные из x509 сертификата. Для этого можно использовать различные команды, которые позволяют получить информацию о сертификате. Вот несколько примеров:

1. **Просмотр содержимого сертификата**:
    ```bash
    openssl x509 -in certificate.crt -text -noout
    ```
    Эта команда отобразит полное содержимое сертификата в читаемом виде.

2. **Извлечение конкретных данных**:
    - **Извлечение Subject (субъекта)**:
        ```bash
        openssl x509 -in certificate.crt -noout -subject
        ```
    - **Извлечение Issuer (издателя)**:
        ```bash
        openssl x509 -in certificate.crt -noout -issuer
        ```
    - **Извлечение даты начала действия (notBefore)**:
        ```bash
        openssl x509 -in certificate.crt -noout -startdate
        ```
    - **Извлечение даты окончания действия (notAfter)**:
        ```bash
        openssl x509 -in certificate.crt -noout -enddate
        ```
    - **Извлечение серийного номера**:
        ```bash
        openssl x509 -in certificate.crt -noout -serial
        ```
    - **Извлечение отпечатка сертификата (SHA1)**:
        ```bash
        openssl x509 -in certificate.crt -noout -fingerprint
        ```

3. **Извлечение открытого ключа**:
    ```bash
    openssl x509 -in certificate.crt -noout -pubkey
    ```

Пример использования на практике:

```bash
openssl x509 -in my_certificate.crt -text -noout
```

Эта команда отобразит полное содержимое сертификата, включая все его поля, такие как Subject, Issuer, даты начала и окончания действия, серийный номер и многое другое.

Эти команды позволят вам получить нужную информацию из x509 сертификата, используя OpenSSL.
