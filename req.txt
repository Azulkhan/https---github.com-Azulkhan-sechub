Чтобы элемент <strong> также появлялся и исчезал вместе с ASPxMemo, вам нужно будет добавить этот элемент на вашу страницу и управлять его видимостью с помощью JavaScript, аналогично тому, как вы управляете видимостью ASPxMemo.

Вот как вы можете это сделать:

Шаг 1: Добавление элемента <strong> на страницу
Добавьте элемент <strong> на вашу страницу ASPX. Для удобства управления видимостью, поместите его в контейнер, например, <div>, и присвойте этому контейнеру идентификатор (ID).

aspx
Copy code
<div id="strongContainer" style="display:none;">
    <strong>Важное сообщение</strong>
</div>
В примере выше, style="display:none;" указывает, что контейнер (и, соответственно, элемент <strong>) изначально будет скрыт.

Шаг 2: Обновление скрипта для управления видимостью
Теперь вам нужно обновить существующий JavaScript-код, чтобы он также управлял видимостью контейнера <div> с элементом <strong>.

html
Copy code
<script type="text/javascript">
    function OnComboChanged(combo) {
        // Проверяем, выбран ли элемент, при котором нужно показать memo и strong
        var showComponents = combo.GetValue() === "ShowMemo";
        
        // Управляем видимостью memo
        var memo = ASPxClientControl.GetControlCollection().GetByName("memo");
        if(memo) {
            memo.SetVisible(showComponents);
        }

        // Управляем видимостью контейнера с <strong>
        var strongContainer = document.getElementById("strongContainer");
        if(strongContainer) {
            strongContainer.style.display = showComponents ? "block" : "none";
        }
    }
</script>
Этот скрипт теперь управляет видимостью как ASPxMemo, так и контейнера с элементом <strong>. Когда пользователь выбирает определенный элемент в ASPxComboBox, оба компонента будут показаны или скрыты в зависимости от выбранного значения.

Такой подход позволяет синхронизировать видимость различных элементов на вашей веб-странице, используя клиентские сценарии, что обеспечивает более динамичное и интерактивное взаимодействие с пользователем.

<dx:ASPxComboBox ID="ASPxComboBox1" runat="server" 
                 AutoPostBack="False" 
                 ClientInstanceName="comboBox"
                 OnClientSelectedIndexChanged="OnComboChanged">
    <Items>
        <dx:ListEditItem Text="Показать Memo и сообщение" Value="ShowMemo" />
        <dx:ListEditItem Text="Скрыть Memo и сообщение" Value="HideMemo" />
    </Items>
</dx:ASPxComboBox>
