Для того чтобы выделить выбранное количество элементов на странице, можно использовать Angular привязку классов ([class.class-name]) или условия (ngClass) в сочетании с CSS-стилями. Вот пример, как это можно сделать:

HTML:
html
Копировать код
<div>
    <label>Результат:
        <span>{{query_report$mob_inv_os_toprint.length - 1}}</span>
        <translate>от</translate>
        <span>{{row.row_num}}</span>
    </label>
    <label>На странице:
        <span [class.active]="perPage === 10" (click)="setPerPage(10)">10</span>
        <span [class.active]="perPage === 25" (click)="setPerPage(25)">25</span>
        <span [class.active]="perPage === 50" (click)="setPerPage(50)">50</span>
    </label> 
    <br>
</div>
TypeScript (Component):
typescript
Копировать код
export class YourComponent {
    perPage: number = 10; // Установите начальное значение

    setPerPage(value: number) {
        this.perPage = value;
        // Другие действия, необходимые при изменении количества элементов на странице
    }
}
CSS:
css
Копировать код
.active {
    font-weight: bold;
    color: blue; /* Можно изменить цвет текста */
    background-color: lightgrey; /* Можно добавить фон */
    border-radius: 5px; /* Пример закругленных углов */
    padding: 5px; /* Пример внутреннего отступа */
}
Объяснение:
HTML: Добавлена директива [class.active], которая добавляет класс active элементу span, если текущее значение perPage совпадает с числом в span.
TypeScript: Добавлена переменная perPage для хранения текущего количества элементов на странице и метод setPerPage, который обновляет это значение.
CSS: Добавлен стиль для класса active, который выделяет выбранный элемент.
Этот подход позволяет динамически изменять стиль выбранного количества элементов на странице и делает пользовательский интерфейс более понятным и удобным.
