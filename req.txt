Для того чтобы найти первый рабочий день месяца (не выходной), можно использовать следующий подход, учитывая, что рабочие дни — это понедельник по пятницу. Мы будем использовать функцию `NEXT_DAY` для того, чтобы найти следующий понедельник, если первый день месяца выпадает на выходной.

Пример запроса:

```sql
WITH first_day AS (
  SELECT TRUNC(SYSDATE, 'MM') AS first_day_of_month
  FROM dual
)
SELECT CASE 
         WHEN TO_CHAR(first_day_of_month, 'D', 'NLS_TERRITORY=RUSSIA') IN ('1', '7') 
         THEN NEXT_DAY(first_day_of_month, 'ПОНЕДЕЛЬНИК')
         ELSE first_day_of_month 
       END AS first_working_day
FROM first_day;
```

Этот запрос сначала вычисляет первый день месяца, а затем проверяет, является ли он выходным (суббота или воскресенье). Если это так, то возвращается ближайший понедельник. В данном случае используется региональные настройки для России, где недели начинаются с понедельника.
