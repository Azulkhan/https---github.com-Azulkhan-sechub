
Чтобы выполнить запрос $http.post в AngularJS с данными в формате XML и использованием прокси, вам следует сделать следующее:

1. Убедитесь, что ваш AngularJS-приложение подключено к модулю $http.
2. Используйте $http.post для отправки POST-запроса.
3. Для отправки данных в формате XML укажите заголовок Content-Type как 'application/xml'.
4. Чтобы использовать прокси, настройте ваш сервер на проксирование соответствующего URL.

Пример:

```javascript
$http.post('http://your-proxy-url.com', data, {
    headers: {
        'Content-Type': 'application/xml'
    }
}).then(function(response) {
    // Обработка успешного ответа
}, function(error) {
    // Обработка ошибки
});
```

Замените `'http://your-proxy-url.com'` на ваш URL-адрес прокси и укажите свои данные XML в переменной `data`.
