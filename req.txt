Похоже, вы хотите получить определенные значения из сертификата, такие как серийный номер и хеш субъекта. Чтобы извлечь эти значения из сертификата с использованием OpenSSL и Lua, вы можете использовать команду `openssl` для извлечения этих данных, а затем обрабатывать их в Lua.

Вот пример скрипта, который выполняет команду `openssl` для извлечения серийного номера и хеша субъекта, а затем обрабатывает вывод, чтобы вывести эти значения:

```lua
-- Функция для чтения вывода команды
local function read_command_output(command)
    local handle = io.popen(command)
    local result = handle:read("*a")
    handle:close()
    return result
end

-- Функция для извлечения серийного номера и хеша субъекта из сертификата
local function extract_certificate_info(certificate_path)
    local info = {}

    -- Команда для получения серийного номера
    local serial_number_command = string.format("openssl x509 -in %s -noout -serial", certificate_path)
    local serial_number_output = read_command_output(serial_number_command)
    info.serial_number = serial_number_output:match("serial=([%x]+)")

    -- Команда для получения хеша субъекта
    local subject_hash_command = string.format("openssl x509 -in %s -noout -subject_hash", certificate_path)
    local subject_hash_output = read_command_output(subject_hash_command)
    info.subject_hash = subject_hash_output:match("(%x+)")

    return info
end

-- Путь к файлу сертификата
local certificate_path = "boi.crt"

-- Извлечение информации о сертификате
local cert_info = extract_certificate_info(certificate_path)

-- Вывод информации о сертификате
if cert_info.serial_number then
    print("Серийный номер сертификата: " .. cert_info.serial_number)
else
    print("Не удалось извлечь серийный номер сертификата")
end

if cert_info.subject_hash then
    print("Хеш субъекта сертификата: " .. cert_info.subject_hash)
else
    print("Не удалось извлечь хеш субъекта сертификата")
end
```

### Объяснение:

1. **read_command_output**: Эта функция выполняет команду и возвращает ее вывод в виде строки.
2. **extract_certificate_info**: Эта функция выполняет команды `openssl` для извлечения серийного номера и хеша субъекта из сертификата.
   - `openssl x509 -in <cert> -noout -serial`: Эта команда извлекает серийный номер сертификата.
   - `openssl x509 -in <cert> -noout -subject_hash`: Эта команда извлекает хеш субъекта сертификата.
3. **certificate_path**: Указывает путь к файлу сертификата.
4. **cert_info**: Содержит извлеченную информацию о сертификате.
5. **Вывод информации**: Проверяется, удалось ли извлечь серийный номер и хеш субъекта, и выводятся соответствующие сообщения.

Этот скрипт поможет вам извлечь и вывести серийный номер и хеш субъекта сертификата, используя Lua и команду `openssl`.
