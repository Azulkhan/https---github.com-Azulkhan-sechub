В Oracle можно использовать функцию `INSTR`, чтобы проверить, есть ли символы после определённого слова в тексте. Например, чтобы проверить, есть ли символы после слова "результат", можно воспользоваться следующим SQL-запросом:

```sql
SELECT CASE 
         WHEN INSTR(LOWER(текст), 'результат') > 0 AND LENGTH(TRIM(SUBSTR(текст, INSTR(LOWER(текст), 'результат') + LENGTH('результат')))) > 0 
         THEN 'Есть символы после слова "результат"'
         ELSE 'Нет символов после слова "результат"'
       END AS проверка
FROM ваша_таблица;
```

Объяснение:
1. `INSTR(LOWER(текст), 'результат') > 0` — находит позицию слова "результат" в тексте (в нижнем регистре).
2. `SUBSTR(текст, INSTR(LOWER(текст), 'результат') + LENGTH('результат'))` — извлекает подстроку, начиная с позиции сразу после слова "результат".
3. `TRIM` удаляет пробелы с начала и конца извлечённой подстроки.
4. `LENGTH(...) > 0` проверяет, есть ли символы в этой подстроке. 

Этот запрос проверит, есть ли какие-то символы (кроме пробелов) после слова "результат".
