Понял. Если вы используете AngularJS и хотите отправить POST-запрос через прокси, вам необходимо настроить прокси на сервере, на котором работает ваше AngularJS-приложение. Затем вы можете отправить запрос через этот прокси, указав нужный URL.

Пример:

```javascript
$http.post('/proxy/your-endpoint', data, {
    headers: {
        'Content-Type': 'application/xml'
    }
}).then(function(response) {
    // Обработка успешного ответа
}, function(error) {
    // Обработка ошибки
});
```

Здесь `/proxy/your-endpoint` - это URL вашего прокси, который настроен на перенаправление запросов на нужный сервер. Вам нужно настроить сервер так, чтобы он перенаправлял запросы на `http://your-other-server.com`.
