Чтобы посмотреть, какие функции доступны в модуле `http`, ты можешь вывести содержимое самого модуля, используя цикл `pairs`. Это поможет отобразить все функции и переменные, которые есть в модуле.

Пример:

```lua
local http = require("pkg/http")

for key, value in pairs(http) do
    print(key, value)
end
```

Этот код выводит все функции и поля, доступные в модуле `http`. Ты сможешь увидеть список функций, которые доступны для использования.

Аналогичный подход можно использовать для любого другого модуля в Lua.
