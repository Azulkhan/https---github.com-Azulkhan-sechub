Для того чтобы посмотреть роли (или привилегии) пользователя в CentOS или любой другой системе на основе Linux, можно использовать несколько команд:

### 1. **Для просмотра текущего пользователя и его групп:**
Используйте команду:
```bash
id
```
Она покажет UID, GID, и все группы, к которым принадлежит пользователь.

### 2. **Просмотр информации о пользователе (его настройках):**
```bash
cat /etc/passwd | grep $(whoami)
```
Эта команда выведет строку с данными о текущем пользователе, включая домашний каталог и оболочку.

### 3. **Для проверки административных прав (sudo):**
Если нужно проверить, есть ли у пользователя права суперпользователя (sudo), можно выполнить:
```bash
sudo -l
```
Эта команда покажет список команд, которые пользователь может выполнять с правами `sudo`.

### 4. **Проверка ролей SELinux (если используется):**
Если у вас настроен SELinux и нужно посмотреть роли в контексте SELinux, выполните:
```bash
semanage user -l
```
Эта команда выведет список пользователей SELinux и их роли.
