В Lua для присвоения содержимого файла переменной можно использовать встроенные функции для работы с файлами, такие как `io.open` и `io.read`. Вот пример:

```lua
-- Открываем файл для чтения
local file = io.open("путь_к_файлу", "r")

-- Проверяем, удалось ли открыть файл
if file then
    -- Читаем содержимое файла
    local content = file:read("*all")
    
    -- Закрываем файл
    file:close()

    -- Присваиваем содержимое переменной
    print(content)  -- выводим содержимое файла
else
    print("Не удалось открыть файл")
end
```

В этом примере:
- `"путь_к_файлу"` — это путь к файлу, который вы хотите открыть.
- `file:read("*all")` читает всё содержимое файла в переменную `content`.
- `file:close()` закрывает файл после чтения.

Таким образом, переменная `content` будет содержать весь текст из файла.
