В JavaScript проверка строки на равенство должна работать корректно, если строки действительно равны и нет пробелов или скрытых символов, которые могут помешать. Вот несколько вещей, которые стоит проверить:

1. **Тип данных**: Убедитесь, что `data.output.cn` является строкой. Можно использовать оператор `typeof` для проверки типа данных.

2. **Пробелы и скрытые символы**: Убедитесь, что нет пробелов или скрытых символов в строке. Можно использовать методы `trim` или `replace` для удаления пробелов.

3. **Точное сравнение**: Используйте оператор `===`, чтобы убедиться, что сравнение происходит без приведения типов.

### Пример проверки строки на равенство:

```javascript
if (typeof data.output.cn === 'string' && data.output.cn.trim() === "CN (2022)") {
    console.log("Match found");
} else {
    console.log("No match");
}
```

### Удаление пробелов и скрытых символов:

```javascript
let cnValue = data.output.cn.trim(); // Убираем пробелы в начале и в конце
if (cnValue === "CN (2022)") {
    console.log("Match found");
} else {
    console.log("No match");
}
```

### Проверка на скрытые символы:

```javascript
let cnValue = data.output.cn.replace(/\s+/g, ''); // Убираем все пробелы
if (cnValue === "CN(2022)") {
    console.log("Match found");
} else {
    console.log("No match");
}
```

### Полный пример с проверкой типа и удалением пробелов:

```javascript
if (typeof data.output.cn === 'string' && data.output.cn.trim().replace(/\s+/g, '') === "CN(2022)") {
    console.log("Match found");
} else {
    console.log("No match");
}
```

Эти шаги помогут убедиться, что строка сравнивается корректно и что нет никаких скрытых символов или пробелов, которые могут помешать правильному сравнению.
