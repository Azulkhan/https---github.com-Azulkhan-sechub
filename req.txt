Для того чтобы исключить последние три значения при итерации через массив в Angular, можно изменить условие в директиве `*ngFor`. Используйте метод `slice` для создания нового массива, исключив последние три элемента.

Пример:

```html
<ng-container *ngFor="let field of fields['Mydatas'].slice(0, fields['Mydatas'].length - 3); let index = index">
  <!-- Ваш код для отображения каждого field -->
</ng-container>
```

Этот подход создает подмассив, который начинается с начала массива и заканчивается тремя элементами раньше, чем конец оригинального массива.
