Конечно, вот примеры команд для открытия файлов в терминале Linux:

### 1. **Чтение текстового файла**

- Если у вас есть файл `example.txt`, и вы хотите вывести его содержимое:

   ```bash
   cat example.txt
   ```

- Для постраничного просмотра:

   ```bash
   less example.txt
   ```

   В `less` можно листать файл с помощью клавиш `Space` (для перехода на следующую страницу) и `q` (для выхода).

- Просмотр последних 10 строк файла:

   ```bash
   tail example.txt
   ```

- Для просмотра файла с обновлением в реальном времени (например, лог-файлов):

   ```bash
   tail -f example.txt
   ```

### 2. **Редактирование текстового файла**

- Открыть файл для редактирования в текстовом редакторе `nano`:

   ```bash
   nano example.txt
   ```

- Открыть файл в редакторе `vim`:

   ```bash
   vim example.txt
   ```

### 3. **Открытие графических файлов**

- Если у вас есть изображение, например, `image.png`, вы можете открыть его с помощью программы по умолчанию для изображений:

   ```bash
   xdg-open image.png
   ```

- Если у вас установлен текстовый редактор с графическим интерфейсом, например `gedit`, вы можете открыть файл так:

   ```bash
   gedit example.txt &
   ```

   Символ `&` в конце команды запускает процесс в фоновом режиме, чтобы терминал остался доступен для других команд.

Эти примеры помогут вам открывать и редактировать файлы прямо из терминала.
