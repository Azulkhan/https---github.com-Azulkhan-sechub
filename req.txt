Для преобразования числа в строку с нужным форматом в PostgreSQL, можно воспользоваться функцией `to_char`. Пример запроса для преобразования числа `128975` в строку `128 975,00` будет следующим:

```sql
SELECT to_char(128975, '999 999,00');
```

Этот запрос преобразует число `128975` в строку `128 975,00`. Если вы хотите применять это преобразование к данным в таблице, например, в поле `my_field` таблицы `my_table`, то запрос будет выглядеть так:

```sql
SELECT to_char(my_field, '999 999,00') AS formatted_field
FROM my_table;
```

Этот запрос вернет значение поля `my_field` в требуемом формате для каждой строки таблицы `my_table`.
