
Чтобы запустить консольное приложение Java на сервере Linux, следуйте этим шагам:

1. **Перенос JAR-файла**: Перенесите JAR-файл вашего приложения на сервер Linux. Вы можете использовать SCP или любой другой метод передачи файлов на удаленный сервер.

2. **Установка Java Runtime Environment (JRE)**: Если на сервере Linux нет JRE, установите его с помощью команды установки пакетов. Например, для Ubuntu:

   ```
   sudo apt update
   sudo apt install default-jre
   ```

   Для других дистрибутивов Linux процесс может отличаться. Убедитесь, что вы используете соответствующую команду для вашего дистрибутива.

3. **Запуск приложения**: Откройте терминал на сервере Linux и перейдите в каталог, где расположен ваш JAR-файл. Затем используйте команду `java -jar ваш_файл.jar`, чтобы запустить приложение. Например:

   ```
   java -jar ваш_файл.jar
   ```

4. **Управление приложением**: После запуска вашего приложения оно будет работать в фоновом режиме. Для остановки приложения вы можете использовать сочетание клавиш `Ctrl + C` в терминале, где оно было запущено.

5. **Настройка автозапуска (опционально)**: Если вы хотите, чтобы ваше приложение автоматически запускалось при загрузке сервера, вы можете настроить его как службу или добавить запуск в файлы инициализации, такие как `rc.local` или `crontab`.

После выполнения этих шагов ваше консольное приложение Java будет успешно запущено на сервере Linux и будет доступно для использования.
