В Linux для проверки существования пути можно использовать команду `test` или ее упрощенную форму `[ ]`. Вот несколько примеров, которые показывают, как проверить наличие файла или директории:

1. Проверка, существует ли файл:
   ```bash
   if [ -f /path/to/file ]; then
       echo "Файл существует"
   else
       echo "Файл не существует"
   fi
   ```

2. Проверка, существует ли директория:
   ```bash
   if [ -d /path/to/directory ]; then
       echo "Директория существует"
   else
       echo "Директория не существует"
   fi
   ```

3. Проверка, существует ли путь (файл или директория):
   ```bash
   if [ -e /path/to/file_or_directory ]; then
       echo "Путь существует"
   else
       echo "Путь не существует"
   fi
   ```

4. Использование команды `test` (аналогично вышеуказанным примерам):
   ```bash
   if test -f /path/to/file; then
       echo "Файл существует"
   else
       echo "Файл не существует"
   fi

   if test -d /path/to/directory; then
       echo "Директория существует"
   else
       echo "Директория не существует"
   fi

   if test -e /path/to/file_or_directory; then
       echo "Путь существует"
   else
       echo "Путь не существует"
   fi
   ```

Эти команды можно использовать в сценариях оболочки (`bash` скриптах) для автоматизации задач, связанных с проверкой существования путей в файловой системе.
