Чтобы изменить размер текста на всей странице, вы можете использовать CSS, применяя стиль к тегу `<body>` или к общим тегам, таким как `<p>`, `<h1>`, и т.д. Вот несколько способов сделать это:

1. **Изменение размера текста для всего содержимого страницы:**

   ```html
   <style>
     body {
       font-size: 18px;
     }
   </style>
   <body>
     <h1>Заголовок</h1>
     <p>Текст на странице будет иметь размер 18 пикселей.</p>
   </body>
   ```

2. **Изменение размера текста для определенных тегов на всей странице:**

   ```html
   <style>
     h1 {
       font-size: 32px;
     }
     p {
       font-size: 16px;
     }
   </style>
   <body>
     <h1>Заголовок</h1>
     <p>Текст абзаца будет иметь размер 16 пикселей.</p>
   </body>
   ```

3. **Использование единиц измерения, зависящих от контекста:**
   
   Вы можете использовать `em`, `rem`, или `%` для задания размера текста в зависимости от контекста или базового размера шрифта.

   ```html
   <style>
     body {
       font-size: 100%; /* эквивалентно 16px в большинстве браузеров */
     }
     h1 {
       font-size: 2em; /* 32px (2 * 16px) */
     }
     p {
       font-size: 1em; /* 16px */
     }
   </style>
   <body>
     <h1>Заголовок</h1>
     <p>Этот текст будет иметь размер 16 пикселей, но его можно изменить изменением базового размера в body.</p>
   </body>
   ```

4. **Использование внешнего CSS-файла для изменения размера текста на всей странице:**
   
   Внешний файл `styles.css`:
   ```css
   body {
     font-size: 18px;
   }
   ```

   Включите его в HTML:
   ```html
   <link rel="stylesheet" href="styles.css">
   <body>
     <h1>Заголовок</h1>
     <p>Текст на всей странице будет иметь размер 18 пикселей.</p>
   </body>
   ```

Эти подходы позволяют вам централизованно управлять размером текста для всей страницы, делая код более управляемым и гибким.
