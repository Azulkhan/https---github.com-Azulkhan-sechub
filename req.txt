Для реализации пагинации в Angular можно использовать компонент `p-table` из библиотеки PrimeNG. Вот пример, как можно это сделать:

1. Установите PrimeNG и необходимые зависимости:

```bash
npm install primeng --save
npm install primeicons --save
npm install @angular/cdk --save
```

2. Импортируйте необходимые модули в ваш модуль приложения (`app.module.ts`):

```typescript
import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';
import { TableModule } from 'primeng/table';
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    BrowserAnimationsModule,
    TableModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }
```

3. Создайте компонент и настройте его для отображения таблицы с пагинацией (`app.component.ts` и `app.component.html`):

```typescript
import { Component, OnInit } from '@angular/core';

interface Product {
  id?: string;
  name?: string;
  price?: number;
  category?: string;
  quantity?: number;
  inventoryStatus?: string;
  rating?: number;
}

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent implements OnInit {
  products: Product[] = [];
  totalRecords: number = 0;
  loading: boolean = true;

  ngOnInit() {
    this.loading = false;
    this.products = this.loadProducts();
    this.totalRecords = this.products.length;
  }

  loadProducts(): Product[] {
    // Загрузите или создайте данные для таблицы
    return [
      { id: '1000', name: 'Product1', price: 100, category: 'Category1', quantity: 10, inventoryStatus: 'INSTOCK', rating: 5 },
      { id: '1001', name: 'Product2', price: 200, category: 'Category2', quantity: 20, inventoryStatus: 'OUTOFSTOCK', rating: 4 },
      // Добавьте больше данных по мере необходимости
    ];
  }
}
```

```html
<!-- app.component.html -->
<p-table [value]="products" [paginator]="true" [rows]="10" [totalRecords]="totalRecords" [loading]="loading">
  <ng-template pTemplate="header">
    <tr>
      <th>Name</th>
      <th>Price</th>
      <th>Category</th>
      <th>Quantity</th>
    </tr>
  </ng-template>
  <ng-template pTemplate="body" let-product>
    <tr>
      <td>{{ product.name }}</td>
      <td>{{ product.price }}</td>
      <td>{{ product.category }}</td>
      <td>{{ product.quantity }}</td>
    </tr>
  </ng-template>
</p-table>
```

Этот пример показывает, как создать таблицу с пагинацией в Angular, используя компонент `p-table` из библиотеки PrimeNG. Таблица будет отображать по 10 записей на странице и поддерживать навигацию по страницам.
