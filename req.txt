Если вы хотите настроить прокси для разработки AngularJS-приложения, обходя политику CORS во время разработки, вы можете использовать средства, такие как `http-proxy-middleware` или `proxy` в вашем `angular.json` или `angular-cli.json` файле.

Вот пример использования `proxy` в файле `angular.json`:

```json
{
  ...
  "architect": {
    "serve": {
      "options": {
        "proxyConfig": "src/proxy.conf.json"
      }
    }
  }
}
```

Затем создайте файл `proxy.conf.json` в вашем проекте:

```json
{
  "/api": {
    "target": "http://your-api-url",
    "secure": false
  }
}
```

Этот конфигурационный файл указывает, что все запросы, начинающиеся с `/api`, будут перенаправлены на `http://your-api-url`. 

После этого, при разработке, все запросы, отправленные с вашего приложения AngularJS на `/api`, будут перенаправлены на ваш API-сервер, обходя политику CORS.
