Для отображения PDF-презентации на HTML-странице есть несколько подходов:

1. **Использование встроенного PDF-просмотра с `<iframe>`:**
   Это самый простой способ. Вы можете вставить PDF-файл на страницу с помощью тега `<iframe>`:
   ```html
   <iframe src="your-presentation.pdf" width="100%" height="600px"></iframe>
   ```
   Этот метод встроит PDF-документ в вашу страницу, и браузер покажет его с помощью встроенного PDF-просмотра.

2. **Использование тега `<embed>`:**
   Подобно `<iframe>`, вы также можете использовать тег `<embed>`:
   ```html
   <embed src="your-presentation.pdf" type="application/pdf" width="100%" height="600px">
   ```

3. **Использование библиотеки PDF.js:**
   PDF.js — это мощная библиотека, которая позволяет отображать PDF-файлы непосредственно на веб-странице, не полагаясь на встроенные механизмы браузера. Это более контролируемый и кастомизируемый способ.

   1. Сначала добавьте PDF.js в проект:
      ```html
      <script src="https://mozilla.github.io/pdf.js/build/pdf.js"></script>
      ```

   2. Затем создайте HTML-элемент, куда будет рендериться PDF:
      ```html
      <canvas id="pdfCanvas"></canvas>
      ```

   3. И используйте следующий скрипт для отображения PDF-файла:
      ```javascript
      var url = 'your-presentation.pdf';

      var loadingTask = pdfjsLib.getDocument(url);
      loadingTask.promise.then(function(pdf) {
          // Получаем первую страницу
          pdf.getPage(1).then(function(page) {
              var scale = 1.5;
              var viewport = page.getViewport({ scale: scale });

              // Подготовка элемента canvas
              var canvas = document.getElementById('pdfCanvas');
              var context = canvas.getContext('2d');
              canvas.height = viewport.height;
              canvas.width = viewport.width;

              // Рендер страницы на canvas
              var renderContext = {
                  canvasContext: context,
                  viewport: viewport
              };
              page.render(renderContext);
          });
      });
      ```

Этот метод даёт больше возможностей для кастомизации и контроля за отображением страниц PDF.

4. **Использование сторонних сервисов:**
   Вы можете использовать сторонние сервисы, такие как Google Docs Viewer, для встраивания PDF:
   ```html
   <iframe src="https://docs.google.com/gview?url=your-presentation.pdf&embedded=true" width="100%" height="600px"></iframe>
   ```

Выбор метода зависит от ваших требований: если вам нужен простой способ, подойдет `<iframe>` или `<embed>`. Если вам нужна полная кастомизация, лучше использовать PDF.js.
