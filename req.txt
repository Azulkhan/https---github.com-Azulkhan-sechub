Для переворота строки в Lua вы можете использовать следующий подход:

```lua
function reverseString(str)
    local reversed = {}
    for i = #str, 1, -1 do
        table.insert(reversed, string.sub(str, i, i))
    end
    return table.concat(reversed)
end

-- Пример использования:
local originalString = "Hello, Lua!"
local reversedString = reverseString(originalString)
print(reversedString)  -- Выведет: "!auL ,olleH"
```

Этот код создаёт функцию `reverseString`, которая принимает строку `str`, разбивает её на символы, сохраняет их в обратном порядке в таблице `reversed`, а затем объединяет их с помощью `table.concat` для формирования результирующей строки.
