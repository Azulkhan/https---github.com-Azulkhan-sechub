Чтобы кнопки были круглыми, а текст внутри них выглядел гармонично, можно использовать следующие стили. Также можно добавить стили для остальных элементов, чтобы они были визуально согласованы с кнопками.

CSS:
css
Копировать код
/* Основной контейнер для выравнивания элементов */
.container {
    display: flex;
    flex-direction: column;
}

/* Стиль для результата */
.result-number {
    font-size: 14px;
    color: #333;
    margin: 0 5px;
}

/* Стили для кнопок */
.button {
    display: inline-flex;
    align-items: center;
    justify-content: center;
    width: 40px; /* Ширина кнопки */
    height: 40px; /* Высота кнопки */
    border: 2px solid transparent;
    border-radius: 50%; /* Круглая форма */
    background-color: #f5f5f5;
    color: #333;
    font-size: 14px;
    cursor: pointer;
    transition: all 0.3s ease;
}

.button:hover {
    background-color: #e0e0e0;
    border-color: #ccc;
}

.button.active {
    background-color: #007bff; /* Цвет фона для активного состояния */
    color: white; /* Цвет текста для активного состояния */
    border-color: #0056b3; /* Цвет границы для активного состояния */
    font-weight: bold; /* Жирный текст для активного состояния */
}

.button:focus {
    outline: none; /* Убираем стандартный контур при фокусе */
}

.button:active {
    background-color: #0056b3; /* Цвет фона при нажатии */
    border-color: #004080; /* Цвет границы при нажатии */
}
Объяснение:
HTML:

Добавлен класс result-number для стилизации чисел результата.
Кнопки с классом button остаются круглыми и активные состояния выделяются.
CSS:

.container: Выравнивает элементы внутри контейнера вертикально.
.result-number: Устанавливает размер шрифта и цвет для чисел результата.
.button: Устанавливает размеры кнопок, делает их круглыми с помощью border-radius: 50%, и добавляет стили для фона, границ и текста.
.button:hover, .button.active, и .button:active: Определяют стили для состояния наведения, активного состояния и нажатия кнопки соответственно.
Эти стили обеспечат визуальную согласованность между элементами и кнопками, а также улучшат пользовательский интерфейс.
