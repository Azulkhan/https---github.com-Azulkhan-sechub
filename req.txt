Если вам нужно увеличить количество строк в результате запроса `SELECT` без вставки новых данных, вы можете использовать технику дублирования строк. Один из способов сделать это - воспользоваться функцией генерации серий и соединением с вашими данными.

Вот пример запроса:

```sql
WITH base_data AS (
  SELECT * FROM your_table LIMIT 18
),
expanded_data AS (
  SELECT 
    base_data.*,
    generate_series(1, 56) AS duplication_factor
  FROM base_data
)
SELECT * FROM expanded_data
LIMIT 1000;
```

Объяснение:
1. CTE `base_data` выбирает 18 записей из вашей таблицы.
2. В CTE `expanded_data` каждая запись из `base_data` дублируется 56 раз с использованием функции `generate_series`.
3. Итоговый запрос выбирает первые 1000 записей из расширенного набора данных.

Замените `your_table` на название вашей таблицы. Этот подход позволяет вам получить 1000 строк в результате `SELECT` без вставки дополнительных данных в таблицу.
