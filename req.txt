Чтобы извлечь значение из XML-документа в Lua скрипте, нужно сначала разобрать XML и затем извлечь нужное значение из соответствующего тега. Lua не имеет встроенных функций для работы с XML, поэтому вам нужно будет использовать библиотеку для парсинга XML. Рекомендуется использовать библиотеку, такую как `LuaXML`, которая позволяет удобно работать с XML в Lua.

Вот пример использования `LuaXML` для извлечения значения из тега XML:

```lua
local xml = require("xml")

-- Ваш XML-документ
local xml_data = [[
    <root>
        <tag1>значение1</tag1>
        <tag2>значение2</tag2>
    </root>
]]

-- Создание объекта XML
local handler = xml.newParser()
local dom = handler:parse(xml_data)

-- Извлечение значения из тега
local value = dom.root.tag1[1]

-- Вывод значения
print("Значение из тега <tag1>: " .. value)
```

В этом примере `xml_data` содержит ваш XML-документ. `LuaXML` используется для разбора этого документа, и затем вы можете получить доступ к значению `<tag1>` через `dom.root.tag1[1]`.
