Чтобы при наведении мыши на ячейку таблицы текст раскрывался полностью, можно использовать комбинацию CSS для создания эффекта всплывающей подсказки (tooltip). Вот как это можно сделать:

1. **HTML**:
```html
<td class="expand-on-hover">{{row.monnum}}</td>
```

2. **CSS**:
```css
.expand-on-hover {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
    max-width: 200px; /* Замените на нужное значение ширины */
    position: relative;
}

.expand-on-hover:hover {
    overflow: visible;
    white-space: normal;
    max-width: none;
    background-color: #fff; /* Цвет фона при наведении, чтобы текст был читаемым */
    z-index: 1;
    border: 1px solid #ddd; /* Можно добавить границу, чтобы текст был лучше виден */
    padding: 5px; /* Можно добавить отступы, чтобы текст не был слишком прижат к границе */
}
```

В этом примере текст в ячейке таблицы будет обрезан с многоточием, если он не помещается в одну строку. При наведении курсора мыши на ячейку, текст будет отображаться полностью.

Эти стили обеспечивают, что текст, который не помещается в одну строку, будет обрезан с многоточием, а при наведении курсора мыши на ячейку он будет отображаться полностью. Вы можете настроить стили под свои нужды, чтобы добиться нужного визуального эффекта.
