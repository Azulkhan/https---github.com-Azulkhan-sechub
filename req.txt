
Для того чтобы в ASPxCalendar можно было выбрать только дату, без указания времени, вам нужно настроить контрол таким образом, чтобы он отображал только календарь для выбора даты. ASPxCalendar является частью ASP.NET DevExpress и предоставляет удобный пользовательский интерфейс для выбора дат.

Пример настройки ASPxCalendar для выбора только даты:

```aspx
<dx:ASPxCalendar ID="ASPxCalendar1" runat="server" >
</dx:ASPxCalendar>
```

В данном случае, `ASPxCalendar` уже по умолчанию настроен на выбор даты, без времени. Если вам нужно обработать выбранную дату на сервере, вы можете использовать событие `SelectionChanged`:

```csharp
protected void ASPxCalendar1_SelectionChanged(object sender, EventArgs e)
{
    // Получаем выбранную дату
    DateTime selectedDate = ASPxCalendar1.SelectedDate;
    // Здесь может быть ваш код для работы с выбранной датой
}
```

Если вам необходимо ограничить диапазон дат, которые можно выбрать, используйте свойства `MinDate` и `MaxDate`:

```aspx
<dx:ASPxCalendar ID="ASPxCalendar1" runat="server" MinDate="2023-01-01" MaxDate="2023-12-31">
</dx:ASPxCalendar>
```

Это ограничит выбор дат в календаре диапазоном от 1 января 2023 года до 31 декабря 2023 года.

Для более тонкой настройки внешнего вида и поведения ASPxCalendar вы можете использовать различные свойства и методы, предоставляемые данным контролом. Обратитесь к официальной документации DevExpress для получения более подробной информации.
