Попробуем другой подход для Oracle SQL, чтобы корректно определить первый рабочий день месяца, исключая выходные (субботу и воскресенье).

Вот правильный запрос:

```sql
WITH first_day AS (
  SELECT TRUNC(SYSDATE, 'MM') AS first_day_of_month
  FROM dual
)
SELECT CASE
         WHEN TO_CHAR(first_day_of_month, 'D', 'NLS_TERRITORY=RUSSIA') = '1' -- Воскресенье
         THEN first_day_of_month + 1
         WHEN TO_CHAR(first_day_of_month, 'D', 'NLS_TERRITORY=RUSSIA') = '7' -- Суббота
         THEN first_day_of_month + 2
         ELSE first_day_of_month
       END AS first_working_day
FROM first_day;
```

В этом запросе:
- Если первый день месяца — воскресенье, добавляется 1 день, чтобы получить понедельник.
- Если первый день месяца — суббота, добавляется 2 дня, чтобы также получить понедельник.
- Если первый день месяца — будний день, он возвращается без изменений.

`NLS_TERRITORY=RUSSIA` используется для правильной настройки начала недели с понедельника.
