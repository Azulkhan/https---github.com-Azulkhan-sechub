To open a file in the terminal in Linux (CentOS or other distributions), you can use various commands depending on what you want to do with the file (view, edit, etc.). Here are some common commands:

### 1. **Opening a file for viewing:**

- **Using `cat`:** Displays the entire content of the file.
  ```bash
  cat filename
  ```

- **Using `less`:** Allows you to scroll through the file. Press `q` to quit.
  ```bash
  less filename
  ```

- **Using `more`:** Similar to `less`, but with less functionality. Also scrolls page by page.
  ```bash
  more filename
  ```

### 2. **Opening a file for editing:**

- **Using `nano`:** Simple text editor in the terminal.
  ```bash
  nano filename
  ```

- **Using `vim`:** Powerful text editor (more complex than `nano`).
  ```bash
  vim filename
  ```
  - To enter **insert mode** in `vim`, press `i`.
  - To **save and exit**, press `Esc`, type `:wq`, then press Enter.

- **Using `vi`:** Similar to `vim`, if `vim` is not installed by default.
  ```bash
  vi filename
  ```
