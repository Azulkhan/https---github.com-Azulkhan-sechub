В Angular есть несколько способов отображения PDF-документов на странице. Вот несколько популярных методов:

### 1. **Использование встроенного HTML `<embed>` или `<iframe>`:**

Это самый простой способ. Можно встроить PDF в страницу с помощью тега `<embed>` или `<iframe>`.

Пример с использованием тега `<embed>`:

```html
<embed src="path/to/your/file.pdf" width="100%" height="600px" type="application/pdf">
```

Пример с использованием тега `<iframe>`:

```html
<iframe src="path/to/your/file.pdf" width="100%" height="600px"></iframe>
```

### 2. **Использование библиотеки `ng2-pdf-viewer`:**

`ng2-pdf-viewer` — это Angular-библиотека, которая позволяет отображать PDF-файлы на странице и поддерживает функции вроде масштабирования, поворота и т.д.

#### Установка:

1. Установите библиотеку:

```bash
npm install ng2-pdf-viewer
```

2. Импортируйте её в ваш модуль:

```typescript
import { PdfViewerModule } from 'ng2-pdf-viewer';

@NgModule({
  declarations: [...],
  imports: [
    PdfViewerModule,
    ...
  ],
  ...
})
export class AppModule { }
```

3. Используйте компонент `pdf-viewer` в шаблоне:

```html
<pdf-viewer [src]="pdfSrc" 
            [render-text]="true"
            style="display: block;"
            [original-size]="true">
</pdf-viewer>
```

4. В вашем компоненте укажите путь к PDF:

```typescript
export class YourComponent {
  pdfSrc = "path/to/your/file.pdf";
}
```

### 3. **Использование библиотеки PDF.js:**

PDF.js — это популярная библиотека для рендеринга PDF в HTML5. Её можно интегрировать в Angular проект.

#### Установка:

1. Установите библиотеки PDF.js:

```bash
npm install pdfjs-dist
```

2. Импортируйте PDF.js в компонент и используйте его для рендеринга PDF.

```typescript
import * as pdfjsLib from 'pdfjs-dist';
import { Component, OnInit } from '@angular/core';

@Component({
  selector: 'app-pdf-viewer',
  template: `<canvas id="pdf-canvas"></canvas>`,
})
export class PdfViewerComponent implements OnInit {
  ngOnInit(): void {
    const url = 'path/to/your/file.pdf';
    const canvas: any = document.getElementById('pdf-canvas');
    const context = canvas.getContext('2d');

    pdfjsLib.getDocument(url).promise.then((pdf) => {
      pdf.getPage(1).then((page) => {
        const viewport = page.getViewport({ scale: 1 });
        canvas.height = viewport.height;
        canvas.width = viewport.width;

        const renderContext = {
          canvasContext: context,
          viewport: viewport,
        };
        page.render(renderContext);
      });
    });
  }
}
```

Этот код загружает и рендерит PDF-документ на `canvas`.

### Заключение:

- Для простого отображения PDF можно использовать теги `<embed>` или `<iframe>`.
- Для более продвинутого управления PDF (масштабирование, поворот и т.д.) можно использовать библиотеку `ng2-pdf-viewer`.
- Если нужен полный контроль над процессом рендеринга PDF, можно использовать библиотеку PDF.js.
