Чтобы реализовать в Excel условное форматирование, которое будет добавлять разные количество нулей в зависимости от длины числа, можно воспользоваться функцией IF и оператором & для объединения текста.

Допустим, у вас числа находятся в колонке A, начиная с ячейки A1. Вы можете использовать следующую формулу в колонке B:

excel
Копировать код
=IF(A1="", "--",
   IF(LEN(A1)=5, "'000(" & A1,
   IF(LEN(A1)=4, "'0000(" & A1,
   IF(LEN(A1)=3, "'00000(" & A1, A1))))
)
Пояснение:
IF(A1=""): Проверяет, пустая ли ячейка. Если ячейка пустая, выводит --.
LEN(A1)=5: Если длина числа в ячейке 5, добавляет в начало '000(.
LEN(A1)=4: Если длина числа 4, добавляет '0000(.
LEN(A1)=3: Если длина числа 3, добавляет '00000(.
Если длина числа не совпадает с условиями, просто выводит само значение.
Эта формула добавляет нужное количество нулей и скобку в зависимости от длины числа, а для пустых ячеек выводит --.

Пример работы:
Для числа 24798 будет результат: '000(24798
Для числа 1234 будет результат: '0000(1234
Для числа 567 будет результат: '00000(567
Если ячейка пустая, результат: --
Протяните формулу вниз для всех строк, и она будет работать с вашим списком автоматически.
