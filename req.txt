Чтобы отформатировать число в Angular, можно использовать Angular Pipe. В данном случае, вам нужен `number` pipe, который позволяет задать формат отображения чисел. Вы можете использовать его в вашем шаблоне следующим образом:

```html
<td>{{ raw.monsum | number:'1.2-2' }}</td>
```

Здесь `number:'1.2-2'` означает, что число будет иметь как минимум одну цифру до запятой, от 2 до 2 цифр после запятой. 

В результате значение `128975` будет отображаться как `128 975,00` (или в зависимости от локали).

Если вам нужна конкретная локаль, например, чтобы использовать запятую в качестве разделителя тысяч и точку в качестве десятичного разделителя, вам нужно будет настроить локаль в вашем приложении.

Вот пример, как настроить локаль в `app.module.ts`:

```typescript
import { registerLocaleData } from '@angular/common';
import localeRu from '@angular/common/locales/ru';

registerLocaleData(localeRu);

@NgModule({
  providers: [{ provide: LOCALE_ID, useValue: 'ru' }]
})
export class AppModule { }
```

Это задаст локаль для всего приложения, и числа будут форматироваться в соответствии с русскими стандартами.
