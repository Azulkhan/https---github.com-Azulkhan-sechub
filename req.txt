
В AngularJS вы можете использовать директиву `ng-model` для связывания значения поля ввода типа файла с переменной в контроллере. Вот как это можно сделать:

```html
<input type="file" ng-model="uploadedFile" />
```

Затем в вашем контроллере AngularJS вы сможете получить доступ к имени файла, загруженного через это поле ввода:

```javascript
app.controller('MainController', function($scope) {
    $scope.uploadedFileName = null;
    
    $scope.getFileDetails = function() {
        if ($scope.uploadedFile) {
            $scope.uploadedFileName = $scope.uploadedFile.name;
        }
    };
});
```

Используйте метод `getFileDetails()` для получения имени файла после того, как пользователь выберет файл.
