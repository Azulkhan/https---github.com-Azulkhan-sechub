Чтобы проверить последние четыре цифры строки в Lua и JavaScript, можно использовать функции для работы со строками.

### Lua
В Lua можно использовать функцию `string.sub` для извлечения последних четырех символов строки и затем проверить их. Вот пример кода:

```lua
local function checkLastFourDigits(value)
    local lastFour = string.sub(value, -4)
    return lastFour
end

local value = "12345678"
print(checkLastFourDigits(value))  -- Output: 5678
```

### JavaScript
В JavaScript можно использовать метод `slice` для аналогичной операции. Вот пример кода:

```javascript
function checkLastFourDigits(value) {
    var lastFour = value.slice(-4);
    return lastFour;
}

var value = "12345678";
console.log(checkLastFourDigits(value));  // Output: 5678
```

В обоих примерах функция `checkLastFourDigits` принимает строку и возвращает последние четыре символа этой строки.
