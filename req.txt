Чтобы отобразить PDF на странице в Angular, можно использовать тег `<iframe>` или библиотеку, такую как `pdf-viewer`. Вот два подхода:

### 1. Использование тега `<iframe>`:
Это самый простой способ отобразить PDF файл:

```html
<iframe [src]="pdfUrl | safe: 'resourceUrl'" width="100%" height="600px"></iframe>
```

В TypeScript-коде:

```typescript
pdfUrl: string = 'assets/sample.pdf';
```

Тут `safe` — это кастомный pipe для безопасной работы с URL. Его можно создать следующим образом:

```typescript
import { Pipe, PipeTransform } from '@angular/core';
import { DomSanitizer, SafeResourceUrl } from '@angular/platform-browser';

@Pipe({ name: 'safe' })
export class SafePipe implements PipeTransform {
  constructor(private sanitizer: DomSanitizer) {}

  transform(url: string, type: string): SafeResourceUrl {
    return this.sanitizer.bypassSecurityTrustResourceUrl(url);
  }
}
```

### 2. Использование библиотеки `ng2-pdf-viewer`:

1. Установите библиотеку:

```bash
npm install ng2-pdf-viewer
```

2. Импортируйте `PdfViewerModule` в ваш модуль:

```typescript
import { PdfViewerModule } from 'ng2-pdf-viewer';

@NgModule({
  imports: [
    PdfViewerModule
    // другие модули
  ],
})
export class AppModule {}
```

3. В компоненте используйте:

```html
<pdf-viewer [src]="pdfUrl" [render-text]="true" style="display: block;"></pdf-viewer>
```

И в TypeScript:

```typescript
pdfUrl: string = 'assets/sample.pdf';
```

Этот способ позволяет вам отображать PDF без необходимости использования внешних плагинов, а также предлагает дополнительные функции, такие как масштабирование, навигация по страницам и т.д.
