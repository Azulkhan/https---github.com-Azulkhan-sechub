Чтобы стили кнопок выглядели современно и привлекательно, можно использовать следующие CSS-стили. Они обеспечат хорошее визуальное представление и взаимодействие с пользователем, включая стили для выделенного состояния и стандартного вида.

HTML:
html
Копировать код

CSS:
css
Копировать код
/* Стили для кнопок */
.button {
    display: inline-block;
    padding: 8px 12px;
    margin-right: 10px;
    border: 2px solid transparent;
    border-radius: 4px;
    background-color: #f5f5f5;
    color: #333;
    text-align: center;
    cursor: pointer;
    font-size: 14px;
    transition: all 0.3s ease;
}

.button:hover {
    background-color: #e0e0e0;
    border-color: #ccc;
}

.button.active {
    background-color: #007bff; /* Цвет фона для активного состояния */
    color: white; /* Цвет текста для активного состояния */
    border-color: #0056b3; /* Цвет границы для активного состояния */
    font-weight: bold; /* Жирный текст для активного состояния */
}

.button:focus {
    outline: none; /* Убираем стандартный контур при фокусе */
}

.button:active {
    background-color: #0056b3; /* Цвет фона при нажатии */
    border-color: #004080; /* Цвет границы при нажатии */
}
Объяснение:
HTML: Применение классов button и active к span элементам. Класс active применяется, когда соответствующее значение perPage совпадает.
CSS:
.button: Стиль для кнопок, включает внутренние отступы, границу, закругление углов и базовые цвета.
.button:hover: Изменение фона и границы при наведении курсора.
.button.active: Специфичный стиль для активного состояния кнопок, включая изменение цвета фона и текста.
.button:focus и .button:active: Стиль для состояния фокуса и нажатия кнопки, чтобы обеспечить хороший пользовательский опыт.
Эти стили помогут сделать кнопки более заметными и приятными для взаимодействия, а также улучшат визуальную и функциональную сторону вашего интерфейса.
