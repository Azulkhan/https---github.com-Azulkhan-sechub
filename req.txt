эВот пример использования цикла `while` в Lua, который будет повторять выполнение, пока заданное значение не будет пустым (nil или пустая строка):

```lua
local value = ""  -- Изначально значение пустое

while value == "" do
    print("Введите значение (пустое для выхода):")
    value = io.read()  -- Читаем ввод пользователя

    if value ~= "" then
        print("Вы ввели: " .. value)
    end
end

print("Цикл завершен.")
```

### Объяснение:
1. **Инициализация:** Переменная `value` инициализируется как пустая строка.
2. **Цикл `while`:** Цикл продолжается до тех пор, пока `value` пустое.
3. **Чтение ввода:** Внутри цикла запрашивается ввод пользователя.
4. **Проверка:** Если введенное значение не пустое, оно выводится на экран.
5. **Завершение:** Цикл завершается, когда пользователь введет непустое значение.
