эЕсли `Mydatas` изначально пустое, можно добавить проверку на наличие элементов в массиве перед тем, как использовать `slice`. Это позволит избежать ошибок, когда массив пустой.

Пример:

```html
<ng-container *ngIf="fields['Mydatas'] && fields['Mydatas'].length > 3">
  <ng-container *ngFor="let field of fields['Mydatas'].slice(0, fields['Mydatas'].length - 3); let index = index">
    <!-- Ваш код для отображения каждого field -->
  </ng-container>
</ng-container>

<ng-container *ngIf="fields['Mydatas'] && fields['Mydatas'].length >= 3">
  <ng-container *ngFor="let field of fields['Mydatas'].slice(-3); let index = index">
    <!-- Ваш код для отображения каждого field -->
  </ng-container>
</ng-container>
```

Эти проверки гарантируют, что код внутри `*ngFor` будет выполняться только в том случае, если массив `Mydatas` существует и содержит достаточно элементов.
