
В Lua нет прямого способа получить список установленных пакетов, как в других языках программирования, например, в Python с помощью `pip list`. Однако, если вы используете Lua в среде, поддерживающей исполнение команд системы, вы можете воспользоваться командой для получения списка установленных пакетов.

Вот пример Lua скрипта, который использует команду `ls` (на Unix-подобных системах) для получения списка файлов и каталогов в текущем каталоге:

```lua
function get_installed_packages()
    local handle = io.popen('ls /path/to/your/packages')
    local result = handle:read("*a")
    handle:close()
    return result
end
```

Этот скрипт открывает процесс командной строки с помощью `io.popen`, выполняет команду `ls`, читает результат и возвращает его. Замените `/path/to/your/packages` на путь к каталогу, где у вас установлены пакеты Lua.

Помните, что этот метод будет работать только на определенных системах и в определенных условиях.
