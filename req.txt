Чтобы создать уникальный стиль для HTML-элементов `input` и `label`, можно использовать CSS. Вот пример:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Custom Styled Input and Label</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
        }
        
        .form-group {
            margin: 20px;
            display: flex;
            flex-direction: column;
            align-items: flex-start;
        }

        label {
            font-size: 1.2em;
            font-weight: bold;
            color: #333;
            margin-bottom: 10px;
            transition: color 0.3s ease;
        }

        input[type="text"], input[type="email"], input[type="password"] {
            padding: 10px;
            border: 2px solid #007BFF;
            border-radius: 5px;
            font-size: 1em;
            transition: border-color 0.3s ease, box-shadow 0.3s ease;
        }

        input[type="text"]:focus, input[type="email"]:focus, input[type="password"]:focus {
            border-color: #0056b3;
            box-shadow: 0 0 10px rgba(0, 123, 255, 0.5);
            outline: none;
        }

        label:hover {
            color: #007BFF;
        }
    </style>
</head>
<body>
    <div class="form-group">
        <label for="username">Username:</label>
        <input type="text" id="username" name="username">
    </div>
    <div class="form-group">
        <label for="email">Email:</label>
        <input type="email" id="email" name="email">
    </div>
    <div class="form-group">
        <label for="password">Password:</label>
        <input type="password" id="password" name="password">
    </div>
</body>
</html>
```

В этом примере используются стили для `label` и `input`, чтобы создать уникальный внешний вид. Элементы `label` имеют более крупный шрифт и изменяют цвет при наведении, а `input` элементы имеют рамку, которая изменяет цвет и добавляет тень при фокусировке. Вы можете настроить стили под свои требования, изменив значения свойств CSS.
