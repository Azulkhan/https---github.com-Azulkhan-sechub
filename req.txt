Для того чтобы получить данные с подсчетом записей или суммами с определенными условиями в PostgreSQL, можно использовать оператор `CASE` в сочетании с агрегатными функциями такими как `COUNT` или `SUM`. Приведу общий пример запроса, который может помочь вам в решении вашей задачи:

```sql
SELECT
    'Наименование 1' AS name,
    COUNT(*) FILTER (WHERE условие1) AS count1,
    SUM(column_name) FILTER (WHERE условие2) AS sum1,
    'Наименование 2' AS name,
    COUNT(*) FILTER (WHERE условие3) AS count2,
    SUM(column_name) FILTER (WHERE условие4) AS sum2,
    -- Добавьте больше строк по аналогии
FROM
    table_name
GROUP BY
    name;
```

Если у вас 11 строк с различными наименованиями и условиями, это можно сделать так:

```sql
SELECT
    'Наименование 1' AS name,
    COUNT(*) FILTER (WHERE условие1) AS count1,
    SUM(column_name) FILTER (WHERE условие1) AS sum1
FROM
    table_name
UNION ALL
SELECT
    'Наименование 2' AS name,
    COUNT(*) FILTER (WHERE условие2) AS count2,
    SUM(column_name) FILTER (WHERE условие2) AS sum2
FROM
    table_name
UNION ALL
-- Добавьте больше строк по аналогии
;
```

Пример использования, если вам нужно получить сумму продаж по разным категориям товаров:

```sql
SELECT
    'Категория 1' AS category,
    COUNT(*) FILTER (WHERE category = 'Категория 1') AS count_category1,
    SUM(sales) FILTER (WHERE category = 'Категория 1') AS sum_sales_category1
FROM
    sales_table
UNION ALL
SELECT
    'Категория 2' AS category,
    COUNT(*) FILTER (WHERE category = 'Категория 2') AS count_category2,
    SUM(sales) FILTER (WHERE category = 'Категория 2') AS sum_sales_category2
FROM
    sales_table
UNION ALL
-- Добавьте больше строк по аналогии
;
```

Этот запрос будет возвращать строки с наименованиями и соответствующими подсчетами или суммами по заданным условиям. Убедитесь, что условия и имена столбцов соответствуют вашей схеме данных.
