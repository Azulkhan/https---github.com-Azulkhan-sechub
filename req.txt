Чтобы апостроф добавлялся только при наличии значения в `sap_barcode`, можно использовать условную конструкцию в Go templates. Например:

```go
{{ if (index .item_value "sap_barcode") }}
    {{ printf "'%s'" (index .item_value "sap_barcode") }}
{{ else }}
    {{ "" }}
{{ end }}
```

Этот код проверяет наличие значения в `sap_barcode` и добавляет апостроф только если значение присутствует. В противном случае ничего не добавляется.
