
Чтобы узнать доступные функции в модуле "crypto", вы можете использовать функцию `pairs` в Lua. Попробуйте использовать `pairs` с таблицей, возвращаемой `require("pkg/crypto")`, чтобы увидеть список всех функций, доступных в этом модуле. Например:

```lua
local crypto = require("pkg/crypto")

for key, value in pairs(crypto) do
    print(key, value)
end
```

Это выведет список всех функций в модуле "crypto".
