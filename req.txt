
Если вы хотите обновить значение переменной из одной функции и использовать его в другой, вы можете воспользоваться сервисом или фабрикой для хранения этой переменной. Вот пример:

```javascript
angular.module('myApp').service('sharedService', function() {
  var sharedVariable = 'initialValue';

  return {
    getSharedVariable: function() {
      return sharedVariable;
    },
    setSharedVariable: function(newValue) {
      sharedVariable = newValue;
    }
  };
});
```

Теперь вы можете использовать этот сервис для обновления и получения значения переменной из различных функций контроллеров:

```javascript
angular.module('myApp').controller('firstController', function(sharedService) {
  sharedService.setSharedVariable('newValue');
});

angular.module('myApp').controller('secondController', function(sharedService) {
  var value = sharedService.getSharedVariable();
  console.log(value); // Выведет 'newValue'
});
```

Таким образом, вы можете обновить значение переменной в одной функции и использовать его в другой.
