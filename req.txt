Если вы хотите создать простой выпадающий список (dropdown) с множественным выбором в AngularJS, используя чистый JavaScript (без создания собственного компонента), вот как это можно сделать:

### Пример HTML:

```html
<!DOCTYPE html>
<html lang="en" ng-app="myApp">
<head>
    <meta charset="UTF-8">
    <title>AngularJS Multiple Select Example</title>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"></script>
</head>
<body>

<div ng-controller="DropdownController">
    <select ng-model="selectedItems" multiple>
        <option ng-repeat="item in items" value="{{ item.code }}">{{ item.name }}</option>
    </select>
    <p>Выбранные элементы: {{ selectedItems }}</p>
</div>

<script>
    // Создаем модуль AngularJS
    var myApp = angular.module('myApp', []);

    // Создаем контроллер AngularJS
    myApp.controller('DropdownController', ['$scope', function($scope) {
        // Пример данных, которые вы можете написать на странице
        $scope.items = [
            { code: '1', name: 'Элемент 1' },
            { code: '2', name: 'Элемент 2' },
            { code: '3', name: 'Элемент 3' },
            { code: '4', name: 'Элемент 4' }
        ];

        // Необязательное начальное значение для выбора
        $scope.selectedItems = ['1', '3']; // Пример начального выбора
    }]);
</script>

</body>
</html>
```

### Объяснение:

- **ng-model="selectedItems"**: Это связывает выбранные элементы из выпадающего списка с переменной `selectedItems` в вашем контроллере AngularJS. Поскольку мы используем атрибут `multiple`, `selectedItems` будет массивом выбранных значений.
  
- **ng-repeat="item in items"**: Директива `ng-repeat` создает опции для выпадающего списка на основе массива `items`.

- **$scope.items**: Это массив объектов, представляющих элементы списка. Вы можете добавить и изменить эти данные, как вам угодно.

- **$scope.selectedItems**: Это массив начальных выбранных элементов. Здесь представлен пример начального выбора элементов с кодами '1' и '3'. Вы можете изменить это значение в соответствии с вашими требованиями.

Этот пример демонстрирует базовый подход к созданию выпадающего списка с множественным выбором в AngularJS, используя чистый JavaScript и HTML.
