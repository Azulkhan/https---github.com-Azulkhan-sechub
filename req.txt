Чтобы передать файл как параметр в Lua скрипте при вызове функции `sendmail`, можно использовать стандартные функции для работы с файлами в Lua. Например, если вам нужно передать содержимое файла, можно открыть и прочитать файл, а затем передать его в функцию.

Пример:

```lua
-- Открываем файл
local file_path = "path/to/your/file.txt"
local file = io.open(file_path, "r")

-- Читаем содержимое файла
local file_content = file:read("*all")

-- Закрываем файл
file:close()

-- Вызываем sendmail с параметром file_content
sendmail(file_content)
```

Если же функция `sendmail` ожидает путь к файлу, то достаточно передать путь:

```lua
local file_path = "path/to/your/file.txt"
sendmail(file_path)
```

Уточните, если в вашей задаче требуется что-то более специфичное, например, работа с файлами другого типа или дополнительная обработка содержимого файла.
