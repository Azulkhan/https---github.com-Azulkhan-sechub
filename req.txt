
Для этого вам нужно использовать промисы (Promises) или асинхронные функции. Вот пример, как это можно сделать:

```html
<div ng-repeat="item in uploads">
    <button ng-click="uploadAndConfirm(item)">Upload</button>
</div>
```

```javascript
$scope.uploadAndConfirm = function(item) {
    item.upload().then(function() {
        confirm();
    });
};
```

Здесь `item.upload()` должен возвращать промис. Когда загрузка завершится успешно, будет вызвана функция `confirm()`.
