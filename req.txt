Для загрузки Forge в AngularJS вам нужно сначала убедиться, что библиотека доступна в вашем проекте. Затем вы можете использовать инструкцию `require` или `import` (в зависимости от того, как вы организуете свой проект) для загрузки и использования Forge. Вот пример того, как это можно сделать:

1. Установите Forge, используя npm или другой менеджер пакетов:

   ```
   npm install @forge/api --save
   ```

2. Создайте файл с именем, например, `app.js`, где будете определять ваш AngularJS модуль и контроллер:

   ```javascript
   // app.js

   // Импортируем необходимые модули из Forge
   const forge = require('@forge/api');

   // Определяем AngularJS модуль
   angular.module('myApp', [])

   // Определяем контроллер и используем Forge в нем
   .controller('myController', function($scope) {
     // Используйте Forge здесь
     // Например:
     forge.util.setCookie('cookieName', 'cookieValue');
   });
   ```

3. Подключите `app.js` в вашем HTML файле, где находится ваше приложение AngularJS:

   ```html
   <!-- index.html -->

   <!DOCTYPE html>
   <html lang="en" ng-app="myApp">
   <head>
     <meta charset="UTF-8">
     <title>AngularJS App</title>
   </head>
   <body>

     <div ng-controller="myController">
       <!-- Ваше содержимое контроллера -->
     </div>

     <!-- Подключаем app.js -->
     <script src="app.js"></script>

   </body>
   </html>
   ```

Обратите внимание, что это пример. В вашем реальном приложении может потребоваться дополнительная конфигурация или обработка ошибок, в зависимости от требований вашего проекта и среды разработки.
