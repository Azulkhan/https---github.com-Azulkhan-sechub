`*ngIf` и `*ngFor` — это две часто используемые структурные директивы в Angular, которые управляют отображением элементов в DOM.

### `*ngIf`

`*ngIf` используется для условного отображения элемента в DOM. Он удаляет элемент из DOM, если условие ложное, и добавляет его, если условие истинное.

Пример:

```html
<div *ngIf="isVisible">This is visible</div>
```

В данном примере `div` будет отображаться только в том случае, если `isVisible` равно `true`.

Вы также можете использовать `else` для отображения альтернативного шаблона, когда условие ложно:

```html
<div *ngIf="isVisible; else noContent">This is visible</div>
<ng-template #noContent>
  <div>This is not visible</div>
</ng-template>
```

### `*ngFor`

`*ngFor` используется для создания элементов на основе коллекции данных. Он итеративно добавляет элементы в DOM для каждого элемента коллекции.

Пример:

```html
<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>
```

В данном примере будет создан элемент `li` для каждого элемента в массиве `items`.

Вы можете использовать дополнительные переменные в `*ngFor`, такие как индекс, последний элемент и так далее:

```html
<ul>
  <li *ngFor="let item of items; let i = index; let last = last">
    {{ i }} - {{ item }} <span *ngIf="last">(last)</span>
  </li>
</ul>
```

В этом примере `i` будет содержать текущий индекс элемента, а `last` будет `true` для последнего элемента в массиве.

### Комбинация `*ngIf` и `*ngFor`

Вы можете использовать `*ngIf` внутри `*ngFor` для условного отображения элементов:

```html
<ul>
  <li *ngFor="let item of items">
    <span *ngIf="item.visible">{{ item.name }}</span>
  </li>
</ul>
```

Также можно использовать `*ngIf` с `ng-container` для применения условного отображения к списку элементов, созданных с помощью `*ngFor`:

```html
<ng-container *ngIf="items.length > 0">
  <ul>
    <li *ngFor="let item of items">{{ item }}</li>
  </ul>
</ng-container>
```

Эти примеры должны помочь вам понять, как использовать `*ngIf` и `*ngFor` для управления отображением элементов в вашем Angular приложении.
