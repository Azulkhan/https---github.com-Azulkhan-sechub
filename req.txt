Чтобы текст вместился в две строки, можно использовать CSS, который позволит перенос текста на следующую строку, но ограничит количество строк. К сожалению, CSS не имеет прямого способа ограничить количество строк в `<td>` элементе. Однако, есть обходной путь с использованием `display: -webkit-box;` и `-webkit-line-clamp`.

Вот пример, как это сделать:

1. **HTML**:
```html
<td class="two-lines">{{row.monnum}}</td>
```

2. **CSS**:
```css
.two-lines {
    display: -webkit-box;
    -webkit-box-orient: vertical;
    -webkit-line-clamp: 2;
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: normal;
    max-width: 200px; /* Замените на нужное значение ширины */
}
```

Это решение использует свойство `-webkit-line-clamp` для ограничения количества строк до двух. 

В случае, если поддержка `-webkit-line-clamp` вам не подходит, вы можете просто позволить тексту переноситься и надеяться, что он будет помещаться в две строки без жесткого ограничения:

```html
<td style="white-space: normal; max-width: 200px;">{{row.monnum}}</td>
```

В этом случае текст будет переноситься на следующую строку, и если он занимает больше двух строк, то будет отображен полностью.
