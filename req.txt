В Go шаблонах (Go templates) используется цикл range для перебора элементов коллекции, такой как массив или карта (map). Если у вас есть коллекция items и вы хотите получить только одно значение, можно воспользоваться условием или задать конкретный ключ для получения нужного элемента. Например:

1. Если вы хотите отобразить одно значение, которое соответствует определенному условию:
go
Копировать код
{{ range $item_key, $item_value := .items }}
  {{ if eq $item_key "desiredKey" }}
    {{ $item_value }}
  {{ end }}
{{ end }}
2. Если вы хотите отобразить элемент с конкретным индексом (например, первый элемент):
go
Копировать код
{{ range $index, $element := .items }}
  {{ if eq $index 0 }}
    {{ $element }}
  {{ end }}
{{ end }}
3. Если вам нужно получить конкретное значение по ключу:
Если у вас есть карта (map), можно обратиться к конкретному ключу напрямую без использования цикла range:

go
Копировать код
{{ $desiredValue := index .items "desiredKey" }}
{{ $desiredValue }}
Эти примеры показывают, как можно извлечь одно значение из коллекции items в Go шаблонах. В зависимости от того, как организованы ваши данные и какое значение вам нужно получить, можно выбрать подходящий способ.
