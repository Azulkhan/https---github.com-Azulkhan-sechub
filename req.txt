Если вы хотите, чтобы таблица Lua автоматически конвертировалась в строку, вам нужно реализовать функцию для преобразования таблицы в строку. В Lua можно использовать рекурсивную функцию для этого:

```lua
function tableToString(tbl)
    local result = "{"
    local first = true
    for k, v in pairs(tbl) do
        if not first then
            result = result .. ", "
        end
        first = false
        if type(k) == "number" then
            result = result .. "[" .. k .. "] = "
        else
            result = result .. k .. " = "
        end
        if type(v) == "table" then
            result = result .. tableToString(v)
        elseif type(v) == "string" then
            result = result .. "\"" .. v .. "\""
        else
            result = result .. tostring(v)
        end
    end
    result = result .. "}"
    return result
end

-- Пример использования
local myTable = { key1 = "value1", key2 = { subKey = "subValue" }, 3, "test" }
print(tableToString(myTable))
```

Эта функция рекурсивно обрабатывает таблицу и ее подтаблицы, корректно форматируя строки и числа.
