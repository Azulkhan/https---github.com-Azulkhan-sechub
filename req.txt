
Вы можете использовать `$window.location.reload()` в AngularJS для перезагрузки текущей страницы. Вот пример:

```javascript
app.controller('MainController', function($scope, $window) {
    $scope.reloadPage = function() {
        $window.location.reload();
    };
});
```

И в вашем HTML:

```html
<button ng-click="reloadPage()">Перезапустить страницу</button>
```
