Вот простой пример использования цикла `for` в Lua:

```lua
-- Цикл for для итерации по числам от 1 до 10
for i = 1, 10 do
    print("Значение i: " .. i)
end

-- Цикл for для итерации по элементам массива
local fruits = {"яблоко", "банан", "вишня"}
for index, value in ipairs(fruits) do
    print("Элемент " .. index .. ": " .. value)
end
```

### Описание:
1. **Первый пример**: Цикл `for` выполняется 10 раз, начиная с `i = 1` и заканчивая `i = 10`. В каждой итерации выводится текущее значение переменной `i`.

2. **Второй пример**: Цикл `for` используется с функцией `ipairs`, которая перебирает массив `fruits`. В каждой итерации выводится текущий индекс и соответствующий элемент массива.
