
Чтобы выбрать первый рабочий день квартала в SQL, вы можете использовать функцию `DATEADD` для добавления дней к началу квартала и функцию `CASE` для проверки, является ли день рабочим. Например, для SQL Server:

```sql
SELECT 
    CASE 
        WHEN DATEPART(dw, DATEADD(qq, DATEDIFF(qq, 0, GETDATE()), 0)) = 1 THEN DATEADD(day, 1, DATEADD(qq, DATEDIFF(qq, 0, GETDATE()), 0)) -- Если первый день квартала приходится на воскресенье, выбираем следующий рабочий день
        ELSE DATEADD(qq, DATEDIFF(qq, 0, GETDATE()), 0) -- В противном случае, просто выбираем первый день квартала
    END AS First_Working_Day_Of_Quarter
```

Здесь `GETDATE()` возвращает текущую дату. Вы можете заменить ее на нужную вам дату.
