
Чтобы добавить логотип и текст в качестве альтернативного текста к `GridViewCommandColumnCustomButton` в `ASPxGridView` и настроить клиентские события, вы можете использовать свойства `Image` и `Text` для кнопки, а также `ClientSideEvents` для определения обработчиков событий на стороне клиента.

К сожалению, напрямую установить альтернативный текст (alt text) для изображения кнопки через свойства `GridViewCommandColumnCustomButton` невозможно, так как это свойство не предоставляется API ASP.NET. Однако, вы можете использовать клиентские события для добавления атрибута `alt` к изображению кнопки после её отрисовки на странице.

### Добавление логотипа

Для добавления логотипа в кастомную кнопку, используйте свойство `Image`:

```aspx
<dx:GridViewCommandColumn VisibleIndex="0">
    <CustomButtons>
        <dx:GridViewCommandColumnCustomButton ID="MyCustomButton">
            <Image Url="~/path/to/your/image.png" />
        </dx:GridViewCommandColumnCustomButton>
    </CustomButtons>
</dx:GridViewCommandColumn>
```

### Клиентские события

Чтобы настроить клиентские события для `ASPxGridView`, вы можете использовать `ClientSideEvents` для определения обработчиков событий. В случае с кастомной кнопкой вам нужно будет использовать события, связанные с гридом в целом, такие как `Init` или `EndCallback`, и затем применить логику к кнопке внутри обработчика события.

Пример добавления атрибута `alt` к изображению кнопки:

```aspx
<dx:ASPxGridView ID="GridView1" runat="server" ClientInstanceName="grid">
    <ClientSideEvents EndCallback="function(s, e) {
        var images = document.querySelectorAll('.dxgv__cci img'); // Класс .dxgv__cci предполагается быть классом изображений внутри кастомных кнопок
        images.forEach(function(img) {
            if (!img.getAttribute('alt')) {
                img.setAttribute('alt', 'Здесь ваш альтернативный текст');
            }
        });
    }" />
    <!-- Конфигурация столбцов и кнопок -->
</dx:ASPxGridView>
```

В этом примере используется событие `EndCallback` грида для добавления атрибута `alt` к изображениям после каждого обновления грида. Вы должны настроить селектор в `querySelectorAll` в соответствии с реальным классом изображений в вашем случае. 

Обратите внимание, что приведенные выше примеры могут потребовать настройки в зависимости от вашей конкретной ситуации и версии используемых компонентов DevExpress.
