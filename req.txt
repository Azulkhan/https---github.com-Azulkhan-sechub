 один пример кода, который могли бы разрабатывать Джон, Мэри и Питер в рамках своего проекта на Lua. Например, функция для генерации случайного числа с использованием встроенной библиотеки Lua `math.random`:

```lua
-- Пример функции для генерации случайного числа в заданном диапазоне
function generateRandomNumber(min, max)
    math.randomseed(os.time())  -- Инициализация генератора случайных чисел на основе текущего времени
    local randomNumber = math.random(min, max)
    return randomNumber
end

-- Пример использования функции
local randomNum = generateRandomNumber(1, 100)
print("Случайное число: " .. randomNum)
```

Этот код содержит функцию `generateRandomNumber`, которая использует `math.random` для генерации случайного числа в заданном диапазоне. Функция также инициализирует генератор случайных чисел на основе текущего времени (через `os.time()`), чтобы обеспечить разнообразие случайных чисел при каждом запуске программы.

Такой подход может быть полезен, например, для создания случайных событий или параметров в игре, что помогает сделать игровой процесс более интересным и разнообразным.
