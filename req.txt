Чтобы сделать кнопку неактивной (disabled) в JavaScript, вы можете использовать свойство `disabled` элемента кнопки. Вот пример кода:

```javascript
// Получаем ссылку на кнопку по её ID
let myButton = document.getElementById('myButtonId');

// Устанавливаем свойство disabled на true, чтобы сделать кнопку неактивной
myButton.disabled = true;
```

Замените `'myButtonId'` на актуальный ID вашей кнопки. После этого кнопка станет неактивной и пользователи не смогут на неё кликать.
