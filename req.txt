Если форматирование числа через Angular Pipe не работает так, как ожидается, можно использовать кастомную функцию для форматирования числа.

Добавьте следующую функцию в ваш компонент TypeScript:

```typescript
formatNumber(value: number): string {
  return value.toFixed(2).replace(/\B(?=(\d{3})+(?!\d))/g, " ").replace('.', ',');
}
```

Эта функция будет форматировать число с разделением тысяч пробелами и заменой точки на запятую.

Затем используйте эту функцию в вашем шаблоне:

```html
<td>{{ formatNumber(raw.monsum) }}</td>
```

Эта реализация должна форматировать число в виде `128 975,00`.
