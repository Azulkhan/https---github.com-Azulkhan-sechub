Чтобы текст внутри `<td>` элемента вместился, можно использовать CSS для управления его отображением. Вот несколько способов, которые могут помочь:

1. **Перенос слов**:
```html
<td style="word-wrap: break-word;">{{row.monnum}}</td>
```

2. **Перенос длинных слов**:
```html
<td style="word-break: break-all;">{{row.monnum}}</td>
```

3. **Сокращение текста с многоточием** (если не нужно показывать весь текст):
```html
<td style="white-space: nowrap; overflow: hidden; text-overflow: ellipsis; max-width: 200px;">{{row.monnum}}</td>
```
В этом случае замените `200px` на нужное значение максимальной ширины.

4. **Перенос текста на несколько строк**:
```html
<td style="white-space: normal;">{{row.monnum}}</td>
```

Выберите тот вариант, который лучше всего подходит для вашей задачи.
