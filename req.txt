Чтобы создать два вложенных цикла `for`, где один цикл перебирает начальников, а другой — их подчинённых, можно организовать данные в виде таблицы, где каждый начальник имеет свой список подчинённых. Вот пример на Lua:

```lua
-- Таблица с начальниками и их подчинёнными
local bosses = {
    {name = "Началь1", subordinates = {"Подчинённый1-1", "Подчинённый1-2"}},
    {name = "Началь2", subordinates = {"Подчинённый2-1", "Подчинённый2-2", "Подчинённый2-3"}}
}

-- Основной цикл по начальникам
for i, boss in ipairs(bosses) do
    print(i .. ", " .. boss.name) -- Выводим номер и имя начальника
    
    -- Вложенный цикл по подчинённым
    for j, subordinate in ipairs(boss.subordinates) do
        print("  " .. subordinate) -- Выводим имя подчинённого с отступом
    end
end
```

Этот код выведет:

```
1, Началь1
  Подчинённый1-1
  Подчинённый1-2
2, Началь2
  Подчинённый2-1
  Подчинённый2-2
  Подчинённый2-3
```

Здесь внешний цикл `for` проходит по начальникам, а вложенный цикл `for` — по их подчинённым. Таким образом, сначала выводится начальник, а затем — его подчинённые.
