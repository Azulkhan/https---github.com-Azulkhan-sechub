Если я правильно понимаю, вы хотите выбрать все записи из таблицы `quer` (`q1`), которые имеют соответствие в `q2` по полю `lm_id`, а также включить в результат столбец `q2.id`. Для этого вам нужно изменить запрос так, чтобы вывод включал оба столбца: `q1.id` и `q2.id`.

Вот как можно модифицировать ваш запрос:

```sql
SELECT q1.id, q2.id
FROM quer q1
LEFT JOIN quer q2 ON q1.lm_id = q2.id
WHERE q2.lm_id = 221;
```

В этом запросе:
- `q1.id` возвращает идентификатор из первой таблицы (`q1`).
- `q2.id` возвращает идентификатор из второй таблицы (`q2`), с которой производится соединение через `q1.lm_id = q2.id`.

Таким образом, результат будет включать два столбца: `q1.id` и `q2.id`.
