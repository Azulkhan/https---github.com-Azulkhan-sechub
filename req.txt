Для получения информации из сертификата X.509 и записи всех данных в один текстовый файл с помощью команды `openssl`, выполните следующие шаги:

1. Скачайте или скопируйте ваш сертификат в файл, например `certificate.crt`.
2. Выполните следующую команду в терминале:

```sh
openssl x509 -in certificate.crt -text -noout > certificate_info.txt
```

Эта команда делает следующее:
- `openssl x509`: команда для работы с сертификатами X.509.
- `-in certificate.crt`: указывает входной файл с сертификатом.
- `-text`: выводит все данные сертификата в читаемом текстовом формате.
- `-noout`: подавляет вывод самого сертификата в base64 формате.
- `> certificate_info.txt`: перенаправляет вывод в файл `certificate_info.txt`.

После выполнения команды, вся информация о сертификате будет записана в файл `certificate_info.txt`.
