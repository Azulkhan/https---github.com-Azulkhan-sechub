Да, можно добавить время к дате, используя операции с датами. Например, чтобы добавить время до конца дня, можно добавить доли дня. В Oracle это можно сделать следующим образом:

```sql
SELECT *
FROM your_table
WHERE your_date_column BETWEEN TO_DATE('2024-07-01', 'YYYY-MM-DD') 
                          AND TO_DATE('2024-07-01', 'YYYY-MM-DD') + INTERVAL '23:59:59' HOUR TO SECOND;
```

Или можно добавить количество дней, соответствующее времени:

```sql
SELECT *
FROM your_table
WHERE your_date_column BETWEEN TO_DATE('2024-07-01', 'YYYY-MM-DD') 
                          AND TO_DATE('2024-07-01', 'YYYY-MM-DD') + 1 - INTERVAL '1' SECOND;
```

Этот запрос добавляет один день к дате и затем вычитает одну секунду, чтобы получить 23:59:59.
