В AngularJS и в простом JavaScript вы можете использовать метод `slice()` для получения последних четырёх символов строки.

### Простой JavaScript

```javascript
let str = "Пример строки";
let lastFourChars = str.slice(-4);
console.log(lastFourChars); // "роки"
```

### AngularJS

В AngularJS вы можете использовать этот же подход внутри контроллера или директивы.

```javascript
angular.module('myApp', [])
  .controller('MyController', ['$scope', function($scope) {
    $scope.myString = "Пример строки";
    $scope.lastFourChars = function() {
      return $scope.myString.slice(-4);
    };
  }]);
```

### Пример использования в HTML с AngularJS

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"></script>
  <script>
    angular.module('myApp', [])
      .controller('MyController', ['$scope', function($scope) {
        $scope.myString = "Пример строки";
        $scope.lastFourChars = function() {
          return $scope.myString.slice(-4);
        };
      }]);
  </script>
</head>
<body ng-controller="MyController">
  <p>Исходная строка: {{myString}}</p>
  <p>Последние четыре символа: {{lastFourChars()}}</p>
</body>
</html>
```

Этот пример показывает, как получить последние четыре символа строки как в простом JavaScript, так и в AngularJS.
