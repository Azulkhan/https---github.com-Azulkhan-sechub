В Oracle вы можете определить первый рабочий день каждого месяца с помощью SQL-запроса, используя функцию `NEXT_DAY` для нахождения ближайшего рабочего дня. Например, если под рабочими днями понимаются будние дни (с понедельника по пятницу), то можно использовать следующий запрос:

```sql
SELECT NEXT_DAY(TO_DATE('01-' || TO_CHAR(SYSDATE, 'MM-YYYY'), 'DD-MM-YYYY') - 1, 'MONDAY') AS first_working_day
FROM dual;
```

Этот запрос:

1. Получает первый день текущего месяца через `'01-' || TO_CHAR(SYSDATE, 'MM-YYYY')`.
2. Использует функцию `NEXT_DAY`, чтобы найти ближайший понедельник после последнего дня предыдущего месяца (через `TO_DATE(...) - 1`).
   
Вы можете заменить `'MONDAY'` на другой день недели, если первый рабочий день может быть не понедельником.
