Чтобы оба изображения стали одинаковыми, можно задать им одинаковую ширину и высоту через CSS. Например, если вы хотите, чтобы оба изображения были размером 250x275, используйте такие стили:

### CSS:

```css
img {
  width: 250px;   /* Одинаковая ширина для обоих изображений */
  height: 275px;  /* Одинаковая высота для обоих изображений */
  object-fit: cover;  /* Масштабирование изображений с обрезкой для сохранения пропорций */
}
```

### HTML:

```html
<img src="image1.jpg" alt="Первое изображение">
<img src="image2.jpg" alt="Второе изображение">
```

### Пояснение:
- `width: 250px` — задает одинаковую ширину 250 пикселей для обоих изображений.
- `height: 275px` — задает одинаковую высоту 275 пикселей.
- `object-fit: cover` — масштабирует изображение так, чтобы оно заполнило блок, сохраняя пропорции, но может обрезать части изображения.

Таким образом, оба изображения будут одинаковыми по размеру, независимо от их исходных пропорций.
