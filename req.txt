
Вот пример JavaScript кода, использующего цикл `while`, чтобы функция выполнялась, когда значение равно 0:

```javascript
let value = 0;

while (value === 0) {
    // Ваш код функции здесь
    console.log("Функция работает");
    value = 1; // Если вы хотите, чтобы функция выполнялась только один раз, вы можете изменить значение переменной value
}
```

Этот код будет выполнять функцию, пока переменная `value` равна 0. Как только `value` будет изменено на что-то отличное от 0, цикл завершится.
