Чтобы извлечь данные из x509 сертификата и записать их в текстовый файл, выполните следующие шаги:

1. **Извлечение данных с помощью OpenSSL**:

    Например, для извлечения полного содержимого сертификата:

    ```bash
    openssl x509 -in certificate.crt -text -noout > certificate_details.txt
    ```

    Эта команда извлечет полное содержимое сертификата и запишет его в файл `certificate_details.txt`.

2. **Извлечение конкретных данных и запись в файл**:

    Для каждого из конкретных данных, как Subject, Issuer и другие, вы можете использовать команду OpenSSL и перенаправить вывод в файл:

    - Извлечение Subject:
        ```bash
        openssl x509 -in certificate.crt -noout -subject >> certificate_details.txt
        ```
    
    - Извлечение Issuer:
        ```bash
        openssl x509 -in certificate.crt -noout -issuer >> certificate_details.txt
        ```

    - Извлечение даты начала действия (notBefore):
        ```bash
        openssl x509 -in certificate.crt -noout -startdate >> certificate_details.txt
        ```

    - Извлечение даты окончания действия (notAfter):
        ```bash
        openssl x509 -in certificate.crt -noout -enddate >> certificate_details.txt
        ```

    - Извлечение серийного номера:
        ```bash
        openssl x509 -in certificate.crt -noout -serial >> certificate_details.txt
        ```

    - Извлечение отпечатка сертификата (SHA1):
        ```bash
        openssl x509 -in certificate.crt -noout -fingerprint >> certificate_details.txt
        ```

    - Извлечение открытого ключа:
        ```bash
        openssl x509 -in certificate.crt -noout -pubkey >> certificate_details.txt
        ```

Пример последовательного выполнения команд для записи всех данных в один текстовый файл:

```bash
openssl x509 -in certificate.crt -text -noout > certificate_details.txt
openssl x509 -in certificate.crt -noout -subject >> certificate_details.txt
openssl x509 -in certificate.crt -noout -issuer >> certificate_details.txt
openssl x509 -in certificate.crt -noout -startdate >> certificate_details.txt
openssl x509 -in certificate.crt -noout -enddate >> certificate_details.txt
openssl x509 -in certificate.crt -noout -serial >> certificate_details.txt
openssl x509 -in certificate.crt -noout -fingerprint >> certificate_details.txt
openssl x509 -in certificate.crt -noout -pubkey >> certificate_details.txt
```

Эти команды помогут вам извлечь нужные данные из сертификата и записать их в текстовый файл.
