Это происходит из-за использования свойства `position: absolute` вместе с `z-index`. Абсолютно позиционированные элементы выводятся вне нормального потока документа, и они могут перекрывать другие элементы страницы.

Чтобы исправить это и убедиться, что второе изображение не перекрывает нижние элементы на странице, можно ограничить область перекрытия второго изображения только его родительским контейнером с помощью свойства `position: relative` и управлением порядком элементов через `z-index`.

### Исправленный пример:

### HTML:
```html
<div class="image-container">
    <img src="image1.jpg" alt="Image 1" class="image image-first">
    <img src="image2.jpg" alt="Image 2" class="image image-second">
</div>
<div class="other-content">
    <!-- Другие элементы страницы -->
    <p>Это текст или другие элементы ниже изображений.</p>
</div>
```

### CSS:
```css
.image-container {
    position: relative;
    width: max-content; /* Размер контейнера под изображения */
}

.image {
    display: block;
    width: 300px; /* Настройте ширину изображений */
}

.image-first {
    position: relative; /* Это изображение остается на месте */
}

.image-second {
    position: absolute;
    top: 75%; /* Сместите вниз */
    left: 50%; /* Сместите вправо */
    transform: translate(-25%, -25%); /* Корректируем положение для наложения */
    z-index: 1; /* Поднятое изображение */
}

/* Стиль для элементов под изображениями */
.other-content {
    position: relative;
    z-index: 0; /* Элементы под изображениями должны быть ниже второго изображения */
}
```

### Что изменилось:
1. **`position: relative` для `.image-container`**: Это ограничивает область, в которой будет позиционироваться второе изображение, внутри родительского контейнера. Это предотвратит его перекрытие элементов, которые идут после контейнера.
2. **`z-index: 0` для `.other-content`**: Это гарантирует, что любые элементы ниже контейнера с изображениями останутся на своем уровне, даже если используется `z-index` для изображений.

Теперь второе изображение будет перекрывать только первое изображение, но не затрагивать остальные элементы страницы.
