В Angular можно последовательно вызвать две функции, используя промисы или async/await, чтобы убедиться, что первая функция завершится до вызова второй. Пример с использованием async/await:

```typescript
async onButtonClick() {
  await this.firstFunction();
  this.secondFunction();
}

firstFunction(): Promise<void> {
  return new Promise((resolve) => {
    // Действия первой функции
    console.log("Первая функция выполнена");
    resolve();
  });
}

secondFunction() {
  // Действия второй функции
  console.log("Вторая функция выполнена");
}
```

В этом примере первая функция возвращает промис, который разрешается после завершения её действий. После этого вызывается вторая функция.
