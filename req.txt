В Angular для создания пагинации обычно используют сторонние библиотеки, такие как ngx-pagination, или реализуют её вручную. Вот пример, как можно использовать библиотеку ngx-pagination для добавления пагинации в ваше Angular приложение:

1. **Установите ngx-pagination**:

   В командной строке выполните следующую команду:
   ```bash
   npm install ngx-pagination --save
   ```

2. **Импортируйте NgxPaginationModule в ваш модуль**:

   Откройте файл модуля (например, `app.module.ts`) и добавьте импорт:
   ```typescript
   import { NgxPaginationModule } from 'ngx-pagination';

   @NgModule({
     declarations: [
       // ваши компоненты
     ],
     imports: [
       // другие модули
       NgxPaginationModule
     ],
     providers: [],
     bootstrap: [AppComponent]
   })
   export class AppModule { }
   ```

3. **Используйте ngx-pagination в вашем компоненте**:

   В компоненте, где вы хотите использовать пагинацию, настройте данные и логику. Например, предположим, что у вас есть компонент `items.component.ts`:

   ```typescript
   import { Component } from '@angular/core';

   @Component({
     selector: 'app-items',
     templateUrl: './items.component.html',
     styleUrls: ['./items.component.css']
   })
   export class ItemsComponent {
     items = []; // массив ваших данных
     p: number = 1; // текущая страница
     
     constructor() {
       // инициализация данных
       for (let i = 1; i <= 100; i++) {
         this.items.push(`Item ${i}`);
       }
     }
   }
   ```

4. **Добавьте пагинацию в шаблон**:

   В файле шаблона вашего компонента (например, `items.component.html`), добавьте следующий код:

   ```html
   <ul>
     <li *ngFor="let item of items | paginate: { itemsPerPage: 10, currentPage: p }">{{ item }}</li>
   </ul>
   
   <pagination-controls (pageChange)="p = $event"></pagination-controls>
   ```

   В этом примере `itemsPerPage` определяет количество элементов на одной странице, а `currentPage` указывает текущую страницу.

Вот и всё! Теперь у вас должна работать пагинация в вашем Angular приложении. Вы можете настроить параметры пагинации, такие как количество элементов на странице и текущую страницу, в соответствии с вашими требованиями.
