Если вам нужно добавить апостроф только в начале значения, вы можете использовать простое условие и конкатенацию строк без использования `printf`. Вот как это можно сделать:

```go
{{ if (index .item_value "sap_barcode") }}
    '{{ index .item_value "sap_barcode" }}'
{{ else }}
    {{ "" }}
{{ end }}
```

Этот код добавляет апостроф только в начале значения `sap_barcode` при его наличии. Если значения нет, ничего не добавляется.
