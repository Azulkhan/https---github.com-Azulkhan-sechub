Для извлечения информации из сертификата `.cer` в Linux и записи её во временный файл с использованием Lua, можно воспользоваться следующими шагами:

1. Извлечь информацию из сертификата с помощью утилиты `openssl`.
2. Записать эту информацию во временный файл с помощью Lua-скрипта.

Примерный сценарий может выглядеть так:

1. **Извлечь информацию с помощью `openssl`:**

   ```bash
   openssl x509 -in сертификат.cer -noout -subject > temp.txt
   ```

   Эта команда извлечет информацию о субъекте из сертификата и запишет её в файл `temp.txt`.

2. **Написать Lua-скрипт для чтения и обработки файла:**

   ```lua
   -- Открываем временный файл для чтения
   local file = io.open("temp.txt", "r")
   
   if not file then
       print("Не удалось открыть файл")
       return
   end

   -- Читаем данные из файла
   local subject = file:read("*a")

   -- Закрываем файл
   file:close()

   -- Обработка или вывод данных
   print("Субъект сертификата: ", subject)

   -- Записываем данные в новый файл
   local out_file = io.open("output.txt", "w")
   out_file:write("Субъект сертификата: ", subject)
   out_file:close()
   ```

   Этот Lua-скрипт:
   - Открывает временный файл `temp.txt`, созданный ранее.
   - Читает данные из файла.
   - Закрывает временный файл.
   - Выводит данные на экран.
   - Записывает данные в новый файл `output.txt`.
