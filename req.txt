Для отображения полного значения поля ввода при наведении курсора мыши можно использовать HTML и CSS. Вот пример, как это можно сделать:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Input Tooltip</title>
    <style>
        .tooltip {
            position: relative;
            display: inline-block;
        }

        .tooltip .tooltiptext {
            visibility: hidden;
            width: auto;
            background-color: black;
            color: #fff;
            text-align: center;
            border-radius: 5px;
            padding: 5px;
            position: absolute;
            z-index: 1;
            bottom: 125%; /* Позиция всплывающего окна */
            left: 50%;
            margin-left: -50px; /* Сдвиг для центрирования */
            opacity: 0;
            transition: opacity 0.3s;
        }

        .tooltip:hover .tooltiptext {
            visibility: visible;
            opacity: 1;
        }
    </style>
</head>
<body>

<div class="tooltip">
    <input type="text" value="Очень длинное значение, которое не помещается полностью">
    <span class="tooltiptext">Очень длинное значение, которое не помещается полностью</span>
</div>

</body>
</html>
```

Этот пример использует HTML и CSS для создания всплывающей подсказки. Когда вы наведете курсор на поле ввода, всплывающая подсказка отобразит полное значение этого поля. Вы можете изменить стиль и позиционирование подсказки по своему усмотрению.
