Чтобы функция `os.getenv("damu_crypto_enabled")` возвращала значение `"1"`, необходимо убедиться, что переменная окружения `damu_crypto_enabled` установлена в значение `"1"`. В Unix-подобных системах это можно сделать следующим образом:

1. В командной строке установите переменную окружения:
   ```sh
   export damu_crypto_enabled=1
   ```

2. Запустите ваш Lua-скрипт из этой же командной строки:
   ```sh
   lua ваш_скрипт.lua
   ```

Если вы используете Windows, вы можете установить переменную окружения так:

1. В командной строке установите переменную окружения:
   ```cmd
   set damu_crypto_enabled=1
   ```

2. Запустите ваш Lua-скрипт из этой же командной строки:
   ```cmd
   lua ваш_скрипт.lua
   ```

В самом Lua-скрипте вы можете проверить значение переменной следующим образом:

```lua
local damu_crypto_enabled = os.getenv("damu_crypto_enabled")
if damu_crypto_enabled == "1" then
    print("Crypto enabled")
else
    print("Crypto not enabled")
end
```

Убедитесь, что переменная окружения установлена до запуска скрипта.
