Если вам нужно добавить апостроф только в начале значения, вы можете использовать простое условие и конкатенацию строк без использования `printf`. Вот как это можно сделать:

```go
{{ if (index .item_value "sap_barcode") }}
    '{{ index .item_value "sap_barcode" }}'
{{ else }}
    {{ "" }}
{{ end }}
```

Этот код добавляет апостроф только в начале значения `sap_barcode` при его наличии. Если значения нет, ничего не добавляется.


Если конструкция `if` не работает, возможно, проблема связана с тем, как значения проверяются или с тем, как шаблон компилируется. Давайте попробуем другой подход, используя функцию `with`, которая выполняет блок только если значение не пустое:

```go
{{ with (index .item_value "sap_barcode") }}
    '{{ . }}'
{{ end }}
```

Эта конструкция будет работать следующим образом:
- Если значение `sap_barcode` существует и не пустое, оно будет вставлено с апострофом в начале.
- Если значение отсутствует или пустое, ничего не будет вставлено.

Попробуйте этот вариант и проверьте, работает ли он в вашем случае.
