
В AngularJS вы можете использовать промисы для выполнения функций по порядку. Для этого вам понадобится зависимость `$q`:

```javascript
item1.upload()
  .then(function() {
    return item2.upload();
  })
  .then(function() {
    $scope.confirm();
  })
  .catch(function(error) {
    console.error('Ошибка:', error);
  });
```

Убедитесь, что `$q` доступен в вашем контроллере или сервисе.
