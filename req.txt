Для записи содержимого сертификата в другой текстовый файл вы можете использовать стандартные возможности Lua для работы с файлами. Вот как это можно сделать:

```lua
-- Функция для чтения вывода команды
local function read_command_output(command)
    local handle = io.popen(command)
    local result = handle:read("*a")
    handle:close()
    return result
end

-- Функция для записи данных в файл
local function write_to_file(filename, content)
    local file = io.open(filename, "w")
    if file then
        file:write(content)
        file:close()
        return true
    else
        return false, "Unable to open file for writing"
    end
end

-- Путь к файлу сертификата
local certificate_path = "boi.crt"

-- Команда для чтения сертификата с помощью openssl
local command = "openssl x509 -in " .. certificate_path .. " -text -noout"

-- Чтение содержимого сертификата
local certificate_content = read_command_output(command)

-- Путь к файлу, в который мы хотим записать содержимое сертификата
local output_file = "certificate_info.txt"

-- Запись содержимого сертификата в файл
local success, error_message = write_to_file(output_file, certificate_content)

if success then
    print("Содержимое сертификата успешно записано в файл: " .. output_file)
else
    print("Ошибка при записи в файл: " .. error_message)
end
```

В этом скрипте:

1. `read_command_output` используется для выполнения команды `openssl` и чтения содержимого сертификата.
2. `write_to_file` открывает файл `certificate_info.txt` для записи и записывает туда содержимое сертификата.
3. После выполнения скрипта, содержимое вашего сертификата будет записано в файл `certificate_info.txt` в текущем рабочем каталоге.

Примечание:
- Убедитесь, что у вас есть разрешения на запись в каталог, где будет создан файл `certificate_info.txt`.
- Путь к файлу сертификата `boi.crt` должен быть корректным и доступным для чтения вашим скриптом Lua.
