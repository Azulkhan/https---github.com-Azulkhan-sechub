Для того чтобы открыть и прочитать данные сертификата в файле `boi.crt` на Linux с использованием Lua, вы можете использовать команду `openssl` через `os.execute` или `io.popen`. 

Команда `openssl` позволяет прочитать и декодировать содержимое сертификата. Вот пример скрипта Lua, который открывает и выводит содержимое файла сертификата:

```lua
-- Функция для чтения вывода команды
local function read_command_output(command)
    local handle = io.popen(command)
    local result = handle:read("*a")
    handle:close()
    return result
end

-- Путь к файлу сертификата
local certificate_path = "boi.crt"

-- Команда для чтения сертификата с помощью openssl
local command = "openssl x509 -in " .. certificate_path .. " -text -noout"

-- Чтение и вывод содержимого сертификата
local certificate_content = read_command_output(command)
print(certificate_content)
```

Этот скрипт выполняет команду `openssl x509 -in boi.crt -text -noout` для чтения и вывода содержимого сертификата в текстовом формате.

Убедитесь, что `openssl` установлен на вашем Linux-системе. Если его нет, установите его с помощью пакетного менеджера, например:

```bash
sudo apt-get install openssl
```

Этот скрипт должен быть выполнен на Linux-системе с установленным `openssl`, и он выведет информацию о сертификате в читаемом текстовом формате.
