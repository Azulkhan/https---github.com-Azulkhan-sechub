

Для того чтобы получить список идентификаторов (ID) строк из `ASPxGridView`, которые были отмечены с помощью чекбоксов, можно использовать несколько подходов. Один из распространённых способов – использовать встроенный функционал `ASPxGridView` для работы с выборкой строк через чекбоксы. Ниже приведён пример того, как это можно сделать.

### Шаг 1: Добавление чекбокса в GridView

Для начала необходимо добавить столбец с чекбоксами в ваш `ASPxGridView`. Это можно сделать с помощью специального типа столбца `GridViewDataCheckColumn` в разметке ASPX:

```xml
<dx:ASPxGridView ID="YourGridView" runat="server" KeyFieldName="ID">
    <Columns>
        <dx:GridViewDataCheckColumn FieldName="Выбран" Caption="Выбран">
            <SelectAllCheckboxMode>Page</SelectAllCheckboxMode>
        </dx:GridViewDataCheckColumn>
        <!-- Другие столбцы -->
    </Columns>
</dx:ASPxGridView>
```

Здесь `KeyFieldName` должен быть установлен в имя поля, которое содержит уникальный идентификатор (ID) для каждой строки.

### Шаг 2: Получение выбранных строк на стороне сервера

Для извлечения ID отмеченных строк можно использовать следующий код на стороне сервера:

```csharp
protected void YourButton_Click(object sender, EventArgs e)
{
    List<object> selectedKeys = YourGridView.GetSelectedFieldValues("ID");
    foreach (object key in selectedKeys)
    {
        // Здесь key – это ID отмеченной строки. Можете использовать его по своему усмотрению.
    }
}
```

В этом примере `YourGridView.GetSelectedFieldValues("ID")` возвращает список значений поля, указанного в `KeyFieldName`, для всех отмеченных строк. Вы можете вызвать этот метод, например, при обработке события нажатия на кнопку, чтобы получить все выбранные ID.

### Важные моменты

- Убедитесь, что у вас включена возможность выбора нескольких строк в `ASPxGridView`. Это можно сделать с помощью свойства `SettingsBehavior.AllowSelectByRowClick` или `SettingsSelection.Enabled`.
- Если вам нужно, чтобы чекбоксы были доступны только для определённых строк, вы можете использовать событие `HtmlRowPrepared` или `HtmlDataCellPrepared` для динамического управления видимостью или доступностью чекбоксов.

Этот подход поможет вам легко получить список ID отмеченных строк для дальнейшей обработки на стороне сервера.
