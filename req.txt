
Понял вас. Если у вас проблемы с использованием функции агрегации вместе с условным выражением `CASE`, вы можете разделить запрос на две части: одна часть для подсчета строк и вторая часть для выбора данных с использованием условного выражения. Например:

```sql
SELECT 
    CASE WHEN total_count = 0 THEN 'пусто' ELSE status END AS status 
FROM (
    SELECT COUNT(*) AS total_count, MAX(status) AS status 
    FROM eaw WHERE it_id = 14440
) AS subquery;
```

Этот запрос сначала подсчитывает общее количество строк и находит максимальное значение столбца `status` для данного условия. Затем внешний запрос использует условное выражение для выбора нужного значения.
