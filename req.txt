В Oracle для выполнения `SELECT` с последующим `INSERT` в другую таблицу можно использовать следующий синтаксис:

```sql
INSERT INTO другая_таблица (колонка1, колонка2, колонка3)
SELECT поле1, поле2, поле3
FROM исходная_таблица
WHERE условия;
```

Пример:

```sql
INSERT INTO employees_archive (employee_id, first_name, last_name)
SELECT employee_id, first_name, last_name
FROM employees
WHERE department_id = 10;
```

Здесь `employees_archive` — это таблица, куда вставляются данные, а `employees` — таблица, из которой берутся поля `employee_id`, `first_name`, `last_name`. Условие `WHERE department_id = 10` фильтрует строки, которые будут вставлены.

Таким образом, вы можете выбрать три поля из одной таблицы и вставить их в другую, применяя фильтры, если нужно.
