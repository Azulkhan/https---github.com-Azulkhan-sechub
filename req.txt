В Lua можно проверить наличие определённых символов в строке, используя функцию `string.find` или метод `string.match`. Вот примеры использования обеих функций:

### Использование `string.find`
Функция `string.find` возвращает начальную и конечную позицию первого вхождения подстроки или паттерна.

```lua
local str = "Hello, world!"
local substring = "world"

local start_pos, end_pos = string.find(str, substring)

if start_pos then
    print("Substring found at position:", start_pos, "to", end_pos)
else
    print("Substring not found")
end
```

### Использование `string.match`
Функция `string.match` возвращает подстроку, соответствующую заданному паттерну, или `nil`, если совпадений не найдено.

```lua
local str = "Hello, world!"
local pattern = "world"

local match = string.match(str, pattern)

if match then
    print("Pattern found:", match)
else
    print("Pattern not found")
end
```

### Пример проверки наличия конкретного символа
Если нужно проверить наличие конкретного символа в строке, можно использовать одну из вышеуказанных функций с паттерном, представляющим этот символ.

```lua
local str = "Hello, world!"
local char = "o"

local found = string.find(str, char)

if found then
    print("Character found")
else
    print("Character not found")
end
```

Этот код проверяет наличие символа "o" в строке `str` и выводит соответствующее сообщение.
