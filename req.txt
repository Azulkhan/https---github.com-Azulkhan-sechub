
Понял вас. Если нет данных, которые нужно выбрать, и вы хотите вернуть слово "пусто", вы можете использовать условное выражение `CASE` в SQL:

```sql
SELECT CASE WHEN COUNT(*) = 0 THEN 'пусто' ELSE status END AS status 
FROM eaw WHERE it_id = 14440;
```

Это вернет "пусто", если нет данных, иначе вернет значение `status`.
