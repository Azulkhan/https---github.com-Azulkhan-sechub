Для создания группировки заголовков в Excel с использованием JavaScript, вы можете воспользоваться библиотекой `xlsx`. С её помощью можно создавать сложные заголовки, объединяя ячейки и группируя их. Вот пример того, как это можно сделать:

```javascript
// Подключаем библиотеку xlsx
const XLSX = require('xlsx');

// Создаем новый рабочий лист
const ws = XLSX.utils.aoa_to_sheet([
  [{ v: 'Общие данные', t: 's', s: { alignment: { horizontal: 'center' } } }, '', { v: 'Данные ПО факту', t: 's', s: { alignment: { horizontal: 'center' } } }, ''],
  ['Номер', 'Класс', 'Номер', 'Класс']
]);

// Объединяем ячейки для заголовков
ws['!merges'] = [
  { s: { r: 0, c: 0 }, e: { r: 0, c: 1 } }, // Объединение ячеек "Общие данные"
  { s: { r: 0, c: 2 }, e: { r: 0, c: 3 } }  // Объединение ячеек "Данные ПО факту"
];

// Создаем новый рабочий лист и добавляем данные
const wb = XLSX.utils.book_new();
XLSX.utils.book_append_sheet(wb, ws, "Sheet1");

// Записываем рабочий лист в файл
XLSX.writeFile(wb, 'example.xlsx');
```

Здесь:

1. Мы используем `XLSX.utils.aoa_to_sheet` для создания рабочего листа из массива массивов (`aoa` - array of arrays).
2. В первой строке массива мы создаем заголовки, используя объект `{ v: 'Значение', t: 's', s: { alignment: { horizontal: 'center' } } }`, чтобы задать значение ячейки, тип (string) и стиль выравнивания (по центру).
3. В `ws['!merges']` мы задаем объединение ячеек, где `s` - начальная ячейка, а `e` - конечная ячейка.

Этот пример создаст Excel файл с двумя строками заголовков, где первые два столбца будут под "Общие данные", а два последних под "Данные ПО факту".
