
В стандартной библиотеке Lua функции `httppost2` нет. Однако, в Lua возможно работать с HTTP-запросами с помощью внешних библиотек, таких как LuaSocket или LuaSec (для HTTPS). Эти библиотеки предоставляют функционал для отправки HTTP-запросов, включая POST-запросы.

Чтобы отправить POST-запрос в Lua, вы можете использовать библиотеку LuaSocket и её модуль `http`. Вот пример кода, который отправляет POST-запрос:

```lua
local http = require("socket.http")
local ltn12 = require("ltn12")

local url = "http://ваш_сайт.com/путь"
local body = "param1=value1&param2=value2" -- данные для отправки
local response = {}

local res, code, response_headers = http.request{
    url = url,
    method = "POST",
    headers = {
        ["Content-Type"] = "application/x-www-form-urlencoded",
        ["Content-Length"] = tostring(#body)
    },
    source = ltn12.source.string(body),
    sink = ltn12.sink.table(response)
}

if code == 200 then
    print(table.concat(response)) -- вывод ответа сервера
else
    print("Ошибка: " .. (code or "неизвестный код"))
end
```

Этот код отправляет POST-запрос с данными `param1=value1&param2=value2` на указанный URL. Ответ сервера собирается в таблицу `response`, которая затем преобразуется в строку для вывода.

Обратите внимание, что перед использованием вам нужно установить LuaSocket. Установка может отличаться в зависимости от вашей системы или среды разработки.
